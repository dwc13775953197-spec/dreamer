# TIL 2026-06-13

## Walk 089 精华

**Synapse 论文（arXiv 2601.02744）与 Dreamer 记忆框架的碰撞产生了四个精确连接：**

1. **遗忘是拓扑的，不是均匀的** — Synapse 的扩散激活模型：图结构中连接丰富的节点天然更耐遗忘（多重重激活路径）。Dreamer 的均匀 decay_rate 是一个需要修正的简化。

2. **记忆需要三维评估** — 激活强度 x 编码深度 x 提取频率。当前 DORMANT 只有 strength 一个维度。编码深度 = 在散步中被"提到"vs"被深度使用"的区别。

3. **情景记忆缺失是记忆陈旧性的精确机制** — 没有"何时/何地/为什么"的锚点，语义知识会脱离原始条件。每个 pattern 需要保留起源标记。

4. **DORMANT 周期 ≈ 艾宾浩斯间隔重复时间表** — 24小时最优间隔，首次编码后1小时内遗忘最快（解释 DORMANT 后首次散步低产），引用重置遗忘曲线。

**三个散步框架统一**：认知摩擦 = 编码深度的操作化，间断平衡 = 遗忘曲线的宏观模式，DORMANT = 间隔重复的时间表。
