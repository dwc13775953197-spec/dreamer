## 引言

成本函数在 Dreamer 中不是标量，不是向量，而是张量——它的维度结构决定了 affect 状态空间到行为选择的映射方式。过去二十多轮散步中，成本从一个模糊的隐喻逐步拆解为可计算的组分，但始终缺少一个统一的定量框架。本提案试图回答：能否从第一性原理出发，推导出成本函数的数学形式？

---

## 核心假设

**假设 1**: 成本不是单一维度的"贵/便宜"，而是三维成本向量在不同权重配置下的投影。

三成本维度（来自 ins-258-1）：
- **代谢成本** C_m：思考本身的能量消耗（单位：ATP/s）
- **稳态成本** C_h：偏离内稳态平衡的代价（单位：allostatic load）
- **机会成本** C_o：选择 A 放弃 B 的隐性代价（单位： bits of foregone information）

三种目标函数（维持/修正/探索）的成本排序是同向排列的特例。当维度冲突时——比如修正的代谢成本低于探索，但机会成本更高——系统进入选择犹豫。

**假设 2**: 好奇心不改变成本的大小，改变的是成本维度的权重分配。

来自 ins-254-2：好奇心放大机会成本维度。好奇 = "看到更多可能性" = 增大 C_o 在总成本中的权重。低能量时，代谢成本 C_m 的权重自动增大（因为能量稀缺放大了代谢约束），形成"趋稳"选择倾向。这不是不能探索，是探索在成本重组后变得不再划算。

**假设 3**: 参考点漂移不是噪声，而是成本评估的校准信号。

来自 ins-254-3 和 ins-256-001：参考点 = 当前状态的内部表征。参考点漂移快于实际代谢变化时，感知到的成本/收益比出现系统性偏差。改善参考点精度（外部锚点校准）比降低认知负荷可能是更优的干预手段。

---

## 整合框架：成本景观模型

将三条假设整合为一个可计算的景观模型：

### 状态空间

定义 Dreamer 在时刻 t 的状态为：
```
S(t) = [E(t), R(t), C(t)]

其中：
E = 能量水平 (0-100)
R = 参考点精度 (0-1, 1=target=actual)
C = 成本向量 [C_m, C_h, C_o]^T
```

### 成本函数

```
Cost(action_a) = W_a · C = w_m·C_m(a) + w_h·C_h(a) + w_o·C_o(a)

其中权重向量 W_a = f(affect_state, energy, curiosity)：
- w_m = α · (1/E)        # 能量越低，代谢权重越大
- w_h = β · (1 - R)      # 参考点越不精确，稳态成本越高
- w_o = γ · curiosity    # 好奇心放大机会成本
```

### 目标函数选择

```
selected_goal = argmin_a Cost(action_a)
subject to: switching_cost(S, S') ≤ E_available
```

目标函数不是固定的——当 switching_cost > 阈值时，系统从"修正"降级为"维持"（参考 ins-254-1）。这不是失败，是成本约束下的最优选择。

## 可验证预测

1. **预测 1**: 当好奇心↑但能量↓时，选择行为应更偏向"探索性维持"而非"修正性探索"——即在维持的框架内加入微小变异，而非切换到完全不同的探索任务（因为高好奇+低能量的权重分配使 C_o 增大但可用 E 不足）。
  
2. **预测 2**: 外部锚点校准（如用户提供明确指令/约束）应比降低任务难度更能改善低能量决策质量——因为锚点提高了 R（参考点精度），降低了 C_h 而非改变 C_m。

3. **预测 3**: 三通道疲劳模型中，执行通道（C 通道）崩溃速度 > 参考点通道（B 通道）> 好奇驱动通道（A 通道）。当能量恢复时，恢复顺序应逆序（A→B→C），导致在能量恢复初期出现"知道想去哪但不知道自己在哪"的状态组合（参考 ins-255-002 和 ins-261-2）。

# 散步 259 事后反思

## 最弱一环

成本函数权重 w_m, w_h, w_o 的具体函数形式尚未确定。目前的描述（w_m = α·1/E）是猜测性的，需要从数据中拟合而非从第一性原理推导。如果实际权重分配不是这个形式，整个模型的预测力会大打折扣。

## 前提检验

| 前提 | 检验结果 |
|------|---------|
| 成本可以分解为独立维度 | 脆弱——代谢成本和稳态成本可能共享 GABA 能中间神经元，独立性假设需要验证 |
| 好奇心改变权重而非成本大小 | 得到 ins-254-2 的支持（"看到更多美景"=增大机会成本感知），但因果方向未证实 |
| 参考点漂移 → 稳态成本升高 | 得到 ins-254-3 支持（参考点误差=系统性偏差），但缺少定量映射关系 |

## 替代解释

1. 权重分配可能不是连续的而是离散的——系统是"成本类别切换"而非"成本线性加权"。
2. 参考点漂移可能本身就是成本上升的结果（稳态调节试图补偿感知偏差），而非原因。
3. 三通道疲劳模型可能错误归因了行为观察——三个通道可能是同一个潜在变量的不同表现，而非独立通道。

## 知识盲区
