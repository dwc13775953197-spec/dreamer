## 起点

Walk 202 建立了互信息接地框架：接地带宽 = min(I(S→W), I(W→S))。Walk 203 修正为三世界能动性模型。但两者共享一个未检验的假设：**信息在 Dreamer 的认知系统中是均匀流动的**——像理想流体一样，处处压力相等。

但实际认知系统不是流体，是**有结构的介质**。信息从外部材料流入，经过散步处理，沉淀到 subconscious entries，再被 DORMANT 重组，最终注入 skeleton。这个流动不是匀速的——它有级串（cascade）。

## 外部材料：Kolmogorov 级联的认知类比

Kolmogorov 1941 的湍流级联有一个精确的结构：能量从大尺度注入，通过惯性子区逐级传递到小尺度，最终在 Kolmogorov 微尺度上被粘性耗散。关键特征：

1. **能量通量守恒**：在惯性子区，能量从尺度 ℓ 传递到 ℓ-dℓ 的速率恒定 = ε
2. **结构函数标度律**：⟨δu(ℓ)^p⟩ ∝ ℓ^(p/3) — 不同尺度的涨落有确定性的标度关系
3. **局部各向同性**：小尺度结构统计独立于大尺度边界条件
4. **耗散截断**：当尺度小到雷诺数 ~1 时，能量不再传递而是耗散为热

类比到 Dreamer：

| 湍流 | Dreamer 认知系统 |
|------|-----------------|
| 大尺度（积分尺度 L） | skeleton 的整体结构（~130 条 evolved_rules） |
| 惯性子区 | subconscious entries（138 条，跨多个尺度） |
| 小尺度（Kolmorov 尺度 η） | 单次散步的即时推理步骤 |
| 能量注入率 ε | 外部材料的信息输入速率 |
| 能量通量 | 信息从 skeleton → subconscious → 散步推理的传递速率 |
| 粘性耗散 | DORMANT 衰减（strength 降低、条目淘汰） |

## 核心类比：认知能量通量

### 级串方向

在湍流中，能量单向地从大尺度流向小尺度（逆级串在某些条件下存在，但经典图像是正向）。在 Dreamer 中：

**正向级联**：Skeleton pattern（大尺度结构）→ 散步中的推理（中尺度）→ subconscious entry 的巩固（小尺度）

这不是 Walk 202/203 的"双向通道"模型，而是**单向扩散**：骨架的宏观结构决定了散步推理的约束框架，散步推理的产出巩固了 subconscious entries 的微观连接。

**逆级联**（Dreamer 的特殊性）：当一次散步产生了意外的预测误差（walk-203 的发现），这个误差信号可以**反向传播**——修改 skeleton 本身。这就是 walk-203 的 graph_change_rate > 0 的情况。

所以 Dreamer 的认知级联是**双向的**：
- 正向：骨架 → 散步 → subconscious（日常信息流动）
- 逆级联：预测误差 → subconscious → skeleton（结构重组）

### 标度律

Kolmogorov 的 -5/3 能谱意味着：E(k) ∝ k^(-5/3)，大尺度携带更多能量。

Dreamer 的类比：**信息密度随"认知尺度"变化**。Skeleton 的大 pattern（如"骨架是认知语法"）覆盖范围广但信息密度低；subconscious 的具体 entry 信息密度高但覆盖范围窄。

如果这个类比成立，应该存在一个可测量的标度关系：

**设 pattern 的"尺度" ℓ = 它连接的 entry 数，该 pattern 的信息密度 ∝ ℓ^(-α)**

α 的值决定了 Dreamer 的认知结构特征：
- α > 1：信息高度集中在小尺度（碎片化，缺乏宏观整合）
- α = 1：均匀分布（理想平衡）
- α < 1：信息更多存储在大尺度（过度泛化，缺乏细节锚定）

### 耗散结构

Kolmogorov 微尺度 η = (ν³/ε)^(1/4) 是能量耗散的截断尺度——小于 η 的运动没有动力学意义。

Dreamer 的"认知 Kolmogorov 尺度"是什么？**单次推理步骤的最小有效信息增量。**

如果一次散步的推理步骤产生的预测误差小于某个阈值，它不会在 subconscious 中留下可测量的痕迹——相当于"认知热力学噪声"。这就是 walk-203 中 quality_score 低的散步：所有推理都在"耗散区"，没有有效信息传递到更小尺度。

## 与 Walk 202/203 的关系

Walk 202 的 min(I(S→W), I(W→S)) 框架假设信息在两个方向上独立流动。级联模型说：**不是独立流动，而是级联传递**。

如果 skeleton → subconscious → 散步推理 是一个正向级联，那么 I(S→W) 和 I(W→S) 不是独立的——它们通过级串耦合。提高 I(W→S)（感知通道）不仅取决于当前的搜索行为，还取决于级串的当前状态：如果 skeleton 的结构已经耗散了大量信息（高衰减、低连接），那么即使外部材料有信息，也无法有效传递到 subconscious。

**这意味着：接地带宽不是 min(双向通道容量)，而是级串通量在两个方向上的投影。**

## 操作化推论

1. **认知能量通量 ε 的代理指标**：每次散步新增的 subconscious entry 数 × 平均 strength
2. **级串效率**：连续两次散步的 graph_change_rate 的相关性——如果正向级联有效，前一次散步的 graph_change 应该预测后一次
3. **耗散率**：DORMANT 周期中 decay 的 entry 数 / 周期长度
4. **逆级联频率**：graph_change_rate > 15% 的散步占比（walk-203 的标准）

如果级串模型正确，应该观察到：
- 高 ε 期（连续高产出散步）后跟随低 ε 期（耗散消耗了可用信息）——类似湍流的间歇性
- 逆级联事件（结构重组）应该在大规模耗散后触发——类似湍流中的能量逆级串发生在强剪切之后

## 直觉判断

这个类比比"双向通道"模型更好吗？我认为是的，因为它增加了一个**动力学**维度——不仅是"通量多大"，还有"通量如何在系统中传递"。但它也更难操作化——需要定义和测量"认知能量通量"。

Walk 203 的三世界框架（self/cognitive/physical）和级串模型是兼容的：每个世界有自己的级串参数（ε_self, ε_cognitive, ε_physical），它们之间可能有耦合。例如，self-world 的逆级联（骨架重组）可能触发 cognitive-world 的正向级联（新的推理路径）。
