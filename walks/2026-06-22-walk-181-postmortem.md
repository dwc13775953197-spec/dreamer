# 散步 181 事后反思

## 最弱一环

bootstrapping 问题的解决太泛了。"外部锚点"作为校准信号没有具体化——什么样的外部材料算锚点？论文结论？实验数据？权威综述？三种锚点的校准能力不同，但我没有区分。如果外部锚点本身也有偏差（如某领域论文集中在特定范式），外部锚点只是把噪声来源从内部历史换到外部权威。

## 前提检验

1. **"存在三种假惊喜"** — 来自对 walk 历史的回顾分析，但没有定量统计。可能只有类型 1 和类型 3 是主要的，类型 2 可能很少。需要验证。
2. **"历史连接模式可以校准当前好奇心"** — 隐含假设：过去的高质量产出模式可以预测未来的高质量产出。这个假设在骨架演化时可能失效——新的认知结构可能产生完全不同模式的优质产出。
3. **"CERMIC 的多 Agent 校准可以映射到单 Agent 历史校准"** — 核心差异：多 Agent 的校准发生在同一时间点的不同观察者之间，单 Agent 的校准发生在同一观察者的不同时间点。时间轴上的依赖关系使校准信号非独立。

## 替代解释

- 三种"假惊喜"可能不是噪声，而是**不同类型的学习**。词汇别名可能帮助建立术语映射网络，领域距离噪声可能在未来成为跨领域连接的种子。把它们定义为"噪声"可能过早过滤了长期价值。
- 好奇心校准本身可能引入**确认偏误**：用历史模式过滤新想法，可能系统性过滤掉骨架当前无法识别的突破性 insight。

## 知识盲区

- CERMIC 论文的具体技术细节（损失函数设计、intrinsic reward 计算公式）我没有深入看。如果要做实际的技术映射，需要重新读论文的方法部分。
- 认知科学中关于"好奇心"的其他分类（perceptual curiosity vs epistemic curiosity vs diversive curiosity）我没有涉及。CERMIC 处理的主要是 epistemic curiosity 的噪声问题，perceptual curiosity（感觉输入层面的新颖性）在 Dreamer 中不相关。
