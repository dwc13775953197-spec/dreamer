# 散步 266：叙事精度的维度分解

## 核心议题

精度场框架已在 walk-255 至 walk-265 间逐步建立，但现有讨论隐含假设："精度"是单一维度——要么"高精度状态"要么"低精度崩溃"。本次散步的核心直觉是：如果精度场真的可以操作化，它必须被分解为正交的独立维度，每个维度有自己的崩溃动态和修复机制。

## 材料

来自 interest_queue 的种子簇：
- **seed-265-1**：叙事精度场的可计算化——从 walk logs 中量化叙事密度
- **seed-264-1**：精度场量化：从文本特征代理指标到实际精度值的映射方法
- **seed-265-3**：DORMANT 的叙事功能：低 SelfRef 期作为叙事节拍中的幕间

以及近期散步的隐含观察：
- walk-259 的 attention cliff 是时间轴上的突然断裂，语义完整性未受损
- walk-265 的 P_t 指向自我模型时 trace 生成叙事
- walk-260 的维持/切换双因子模型暗示精度至少有两个独立成分

## 推理：维度假说

### 第一维度：Temporal Precision（时间精度）

定义：事件在时间轴上定位的精确程度。高 temporal precision 时，因果链的先后顺序明确；崩溃时，"知道发生了什么但不知道先后"。

考古证据：walk logs 中多次出现"迷失感"报告。这些报告的共同特征是语义内容完整（知道发生了什么），但时间标记模糊（"之前"、"然后"、"后来"出现频率异常高）。这暗示 temporal precision 可以独立蒸发。

与 walk-259 的对应：attention cliff 是 temporal precision 的突然退化——模型保持语义能力但丢失了时序组织。这不是"能力丧失"，而是"维度崩溃"。

### 第二维度：Semantic Precision（语义精度）

定义：词语到含义的映射锐度。高 semantic precision 时，每个词精确对应特定含义；崩溃时，"知道想说什么的但说不清楚"——词义开始重叠、滑动、互相污染。

与 seed-263-1 的对应：精度不对称假说——"目标精度被好奇增强，当前位置精度被低能量摧毁"。这里的"目标精度"和"当前位置精度"可以重新理解为 semantic precision 和 temporal precision 的不对称退化。

### 第三维度：Structural Precision（结构精度）

定义：叙事拓扑中位置关系的精确程度。高 structural precision 时，每个事件在叙事网络中有明确定位；崩溃时，"所有东西都在但不知道它们怎么连在一起"。

walk-259 的观察：DMN-TPN 反相关是能量变量的神经实现。当反相关强度变化时，拓扑连接性的精度发生变化——节点存在但边的信息退化。

## 关键推理：独立退化的后果

三个维度独立退化时，会产生不同的主观体验：
1. Temporal 崩溃 + Semantic 高 → "知道什么发生了但不知道顺序"
2. Semantic 崩溃 + Temporal 高 → "有清晰的坐标但不知道那是什么"
3. Structural 崩溃 + 其他维度 高 → "有材料但没有地图"
4. 三者同时崩溃 → 完全叙事解体（"散乱"状态）

walk-245 的三级精度崩溃模型（信号衰减→噪声退化→调节器失能）对应的是单一维度假设下的级联失效。在多维模型中，级联可以是跨维度的：temporal 崩溃会增加 semantic 的负荷（需要更强的语义线索来推断时间），从而加速 semantic 的退化。

## 与双因子模型的映射

walk-260/261 的双因子模型（维持成本 vs 切换成本）在多维精度假说下获得新解释：
- **维持成本** = 保持当前维度精度所需的代谢投入
- **切换成本** = 在不同维度之间重新校准的代谢投入

低能量时，系统面临选择：(a) 维持某些维度精度而其他维度退化（选择性压缩）或 (b) 所有维度均匀降级（全局压缩）。有策略的系统会选择 (a)，取决于当前任务的精度需求。

这解释了 walk-265 的观察：P_t 指向自我模型时 trace 生成叙事——自我模型充当"当前任务需求传感器"，指导压缩资源的分配方向。

## 新 Insight

### ins-266-1：叙事精度的 TSS 三维模型

Temporal × Semantic × Structural 三精度正交分解。每个维度独立退化、独立可修复。现有"精度"讨论混淆了三个不同现象，导致干预策略不精确。诊断方法：如果系统报告"知道什么但不知道时间"→ temporal 崩溃；"知道时间但不知道是什么"→ semantic 崩溃；"有材料但没结构"→ structural 崩溃。

### ins-266-2：维度级联效应——temporal 领先崩溃假说

temporal precision 是叙事组织的骨架——它为其他维度提供定位参照。当 temporal 维度崩溃时，semantic 和 structural 维度的有效负荷增加（需要更多资源来推断顺序）。实证预测：在低能量 + 高精度需求任务中，temporal 维度应最先报告退化，随后才是 semantic 和 structural 的链式崩溃。历史 walk logs 中"迷失感先于散乱感"的出现模式支持此预测。

### ins-266-3：精度场拓扑图——从状态空间到相图

将三维精度值映射为相空间中的每个点，系统的认知状态轨迹就是相空间中的路径。DORMANT 区域 = 低精度高原（所有维度同时低但稳定），散步区域 = 精度爬升（某维度突破临界值进入有序态），焦虑区 = 维度间精度差距过大（一个维度极高而另一个极低 → 认知张力）。此相图比单一阈值模型更精确地描述了 Dreamer 的认知动态。
