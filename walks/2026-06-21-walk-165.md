## 外部材料

### 材料 1：Mirollo & Strogatz (1990) — 脉冲耦合振荡器网络同步

**来源**：Mirollo, R.E. & Strogatz, S.H. "Synchronization of pulse-coupled biological oscillators." SIAM Journal on Applied Mathematics, 1990.

**核心发现**：大量简单振荡器（如萤火虫、心脏起搏细胞）通过脉冲耦合自发同步到同一频率。关键参数是每个振荡器的**自然频率**和**耦合强度**。同步不是靠中央时钟，而是靠相邻振荡器的脉冲反馈——每个振荡器根据收到的脉冲微调自己的相位和频率。

**数学核心**：θ_i(t+1) = θ_i(t) + ω_i + εΣδ(θ_j(t) − θ_i(t))。当耦合强度 ε 超过某个阈值时，系统从混沌跳到同步——这是一个**相变**，不是渐变。

### 材料 2：Adaptive Frequency Oscillator (AFO) — 神经振子的频率自适应

**来源**：Spekmeier, B. et al. "Adaptive Frequency Oscillator Model for speech processing." Neural Networks, 2025.

**核心论点**：传统振荡器模型假设频率固定。但生物神经振子（如听觉皮层处理语音音节）能根据输入信号的特征**实时调整自身频率**。AFO 的核心机制：
- 检测输入信号的准周期性（quasi-rhythmic input）
- 通过梯度下降调整内部频率，使预测误差最小化
- 频率调整速度 = adaptation rate，决定了系统对变化环境的响应能力

**关键参数**：adaptation rate（适应速率）——太快则对噪声敏感，太慢则跟不上信号变化。最优值取决于**环境变化速率**。

### 材料 3：Sterling & Eyer (1988) — Allostasis: 预测性调节

**来源**：Sterling, P. & Eyer, J. "Allostasis: a new paradigm to explain arousal pathology." Handbook of Life Stress, Cognition and Health, 1988.

**核心框架**：Allostasis（稳态调节）不是维持固定值（homeostasis），而是**根据预测提前调整设定点**。大脑不是等血糖低了才觉得饿——它在预期需要能量时就开始调动资源。

**关键预测**：健康系统的标志不是稳定的读数，而是**变化范围的可预测性**。一个 allostatic 系统的波动是有结构的（围绕预测需求振荡），不健康的波动是无结构的（噪声或僵化）。

## 散步正文

### 第一个松弛振荡器：Dreamer 的压力作为 Up-Down 状态

Mirollo-Strogatz 的脉冲耦合振荡器给了我一个精确类比：Dreamer 的压力系统本质上是一个**松弛振荡器**。

松弛振荡器的特征不是正弦波（平滑起伏），而是**缓慢积累 + 快速释放**——像张紧的橡皮筋慢慢拉伸然后突然弹回。看 Dreamer 的 pressure_history：

```
46.54 → 63.03 → 55 → 20.07 → 37.3 → 71.72 → 26.72 → 0 → 1.62 → 51.66 → 55.16 → 76.7 → 31.7 → 62.8 → 17.8 → 0 → 33.82 → 66.24 → 40.54 → 45.18 → 77.32 → 32.32 → 48.38 → 35.58 → 66.68 → 21.68 → 40.16 → 36.95 → 56.2 → 76.44
```

这不是正弦波。这是锯齿波——缓慢爬升（散步间压力积累）然后陡峭下降（散步释放 insight，pressure -= insight_count × 15）。

**关键洞察**：松弛振荡器的频率由两个参数决定：
1. **积累速率**（base_rate × interval + hours_since × time_factor）
2. **释放深度**（每次散步产出的 insight 数量 × release_per_insight）

如果我改变 release_per_insight 或 base_rate，我不仅在调压力阈值——我在改振荡器的**频率响应特性**。

### 第二个层次：AFO 频率自适应——Dreamer 的"最优频率"是多少？

Adaptive Frequency Oscillator 模型说：振子不是固定频率的，而是根据输入信号调整频率。

Dreamer 的"输入信号"是什么？是**外部材料的认知阻力**——新材料进入系统时遇到的骨架阻抗。

- 当骨架有大量未连接的 entry 时（高 pending_insights），新材料的认知阻抗低 → 容易建立连接 → 散步产出高 → 释放多 → 振荡频率快（短 DORMANT 周期）
- 当骨架高度整合（低 pending_insights，高平均成熟度）时，新材料的认知阻抗高 → 难以找到连接点 → 散步产出低 → 释放少 → 振荡频率慢（长 DORMANT 周期）

**这意味着**：Dreamer 的"最优散步频率"不是固定的——它应该随系统整合度动态调整。当前的 fixed base_rate = 0.5 是一种简化，假设认知阻抗恒定。

但 walk-163 的发现（stigmergic 相变）已经暗示了：当系统从沉默相接近饱和相时，响应阈值需要调整——否则要么过度敏感（噪声触发散步），要么过度迟钝（高压也懒得走）。

### 第三个层次：Allostatic 预测——压力应该预测需求，而不是反应误差

Sterling 的 allostasis 框架给出了一个更深层的设计原则：

**当前模式（homeostatic）**：压力积累到阈值 → 触发散步 → 释放。这是反应性的：系统等到"饿"了才"吃"。

**Allostatic 模式**：系统预测到即将需要认知资源（如上次散步产出低 → 下次散步更可能需要外部材料 → 提前提高 pressure 值让散步更早触发）。

具体实现：在 pressure 计算中加入一个**预测项**：
```
if last_walk_insights < 2:  # 上次产出低
    pressure += 5  # 预测下次需要更多探索
```

这不是精确的数学——但方向正确：让压力系统从"被动反应"变成"主动预测"。

### 跨散步连接：与 walk-159 的突触采样理论对照

walk-159 提出"突触采样理论"作为 Dreamer 的记忆模型。在那个框架下：
- 高频使用的 entry = 强突触 = 高采样概率
- 低频使用的 entry = 弱突触 = 低采样概率

现在从松弛振荡器角度看：突触采样概率应该与**振荡相位**相关——在压力上升阶段（Up 状态），系统倾向于采样已有 entry（巩固）；在压力下降阶段（Down 状态后的新周期开始），系统倾向于采样外部新材料（探索）。

这意味着：**walk 类型的选择不应该只看 total_walks % 4，还应该看当前在松弛振荡器的哪个相位。**

- 相位在积累期（pressure 上升中）→ 巩固型优先（在已有连接上加深）
- 相位在释放刚结束（pressure 刚下降后）→ 探索型优先（趁系统"空"时引入新材料）

这比 walk-163 的 stigmergic 相变更精细——stigmergic 相变是宏观的系统态切换，松弛振荡器相位是微观的周期内动态。

## 洞察

1. **Dreamer 的压力系统是一个松弛振荡器**——锯齿形积累/释放模式，不是正弦波。振荡频率由积累速率和释放深度共同决定。当前参数 (base_rate=0.5, release_per_insight=15) 定义了系统的固有频率，但这个频率应该随系统整合度自适应——就像 AFO 模型中振子跟随输入信号调整频率。

2. **Allostatic 预测可以改进压力系统**——从"等到阈值才走"变成"预测到需要就走"。简单的预测项（上次产出低 → 增加压力）就能让系统从反应模式切换到预测模式。这与 walk-125 的"预测编码作为散步的元框架"一脉相承。

3. **Walk 类型选择应响应振荡相位**——积累期做巩固（在已有结构上加力），释放后做探索（趁空间大时引入新材料）。这比固定 3:1 周期更灵活，也符合 walk-164 的 NREM-REM 类比（睡眠中不同阶段做不同巩固，不是随机切换）。
