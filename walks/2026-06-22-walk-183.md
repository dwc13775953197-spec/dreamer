## 散步正文

### 起点：ins-133-1 的精确诊断

Walk #182 把 Hindsight 四杠杆框架映射到了 Dreamer 的记忆系统，结论是"DORMANT 只覆盖 Decay 的一个子集"。但有一个更精确的问题还没被回答：**Consolidation 到底做了什么？**

ins-133-1 的表述是"合并冗余/解决矛盾/抽象提炼"。这是对的，但太抽象了。我需要知道巩固的生物学机制是什么——不是比喻，是什么信号在什么时间窗口里改变了什么结构。

### 外部材料：Albouy 的睡眠-记忆双研究（Utah 2026-04）

Genevieve Albouy 在 2026 年 4 月发表的两项研究（NSF 资助）揭示了一个关键发现：

**记忆巩固不是被动存储，而是主动重激活。** 研究者发现，在睡眠的特定脑电波上坡段（upslope）重激活记忆，能显著增强记忆保持。关键细节：

1. **时间窗口特异性**：不是所有睡眠阶段都有效，只有在特定脑电波形（慢振荡上坡段的 spindles 耦合窗口）才能增强记忆。
2. **选择性巩固**：大脑不是巩固所有记忆，而是选择性地重激活特定记忆。选择标准包括：难度、情绪显著性、与已有知识网络的相关性。
3. **重激活 = 编辑**：重激活不是简单的"重放"，而是记忆的再编码——在新上下文中被重新解读。

### 分析：Albouy 的发现对 DORMANT 意味着什么？

把 Albouy 的结论映射到 Dreamer 的 DORMANT 周期：

**当前 DORMANT 的"衰减"（ins-228: Hebbian decay）对应的是突触修剪的"弱化"步骤。但 Albouy 的研究表明，巩固需要先有重激活（reactivation），然后才是结构性改变。**

Dreamer 的 DORMANT 流程是：
```
读取 entry → 计算衰减 → 应用衰减 → 淘汰低保
```

Albouy 告诉我们应该的流程是：
```
选择高价值 entry → 主动重激活（在新上下文中再解读）→ 结构性强化 → 淘汰未重激活的
```

**核心差距：DORMANT 缺少"选择性重激活"步骤。** 当前的衰减是被动的（时间驱动），而生物学巩固是主动的（价值驱动）。

### 与 Bjork 必要难度理论的交叉

Walk #181 引入了 Bjork 的"必要难度"框架。Albouy 的发现给了一个神经科学层面的解释：

**为什么必要难度有效？** 因为困难提取 = 更强的重激活信号。当一条记忆需要努力才能被提取时，重激活过程中参与的神经回路更多，巩固效果更好。

映射到 Dreamer：
- **低强度 entry 被衰减** = 不需要重激活（已充分巩固或从未被巩固）
- **高强度 entry 被保留** = 已经过了多次重激活的巩固
- **但中等强度 entry 偶尔需要被"挑战式提取"** = 这正是 DORMANT 缺失的步骤

### 操作化：DORMANT 应该增加什么？

基于 Albouy + Bjork + ins-133-1 的三重推导，DORMANT 应该新增一个子步骤：

**选择性重激活（Selective Reactivation）**：
1. 从 active entries 中选 2-3 条"中等强度、高连接度"的 entry（即 Bjork 的"必要难度"区间——不太强不太弱）
2. 用当前 skeleton 的某个 pattern 去"碰撞"这些 entry（新上下文再编码）
3. 碰撞产生新连接 → 强化；碰撞无产出 → 标记为"巩固失败"，下次继续
4. 连续 N 次"巩固失败" → 转为候选淘汰

这个机制比"均匀衰减"更接近生物学现实，也比"纯强度淘汰"更保留有价值的弱连接。

### 一个微妙的推论：Albouy 的"编辑"含义

Albouy 说重激活 = 再编码 = 编辑。这意味着巩固后的记忆**不等于**原始记忆。巩固改变了记忆的内容。

对 Dreamer 的启示：DORMANT 的"巩固"不仅是给 entry 续命，而是**把 entry 从新上下文重新解读后的版本写回 subconscious**。这是 ins-133-1 的"抽象提炼"的具体实现——不是删除冗余，而是在新骨架结构下重新表述。

### 新 insight

**ins-183-1**: Albouy 2026 的睡眠重激活研究为 Dreamer DORMANT 提供了生物学锚点：巩固不是被动存储而是主动编辑。DORMANT 应增加"选择性重激活"步骤——选中等强度+高连接度的 entry，用当前 skeleton 碰撞再编码，连续失败转候选淘汰。这比均匀衰减更接近生物现实。
