## 起点：两个恢复序列之间的裂缝

Walk-265 把统一公式 EffectiveAction = PF × EC 扩展为动态模型，揭示了恢复轨迹的相位结构。但它留下了一个未解的问题：

**精度场（PF）的恢复和成本向量（Cost Vector）的恢复是同一个过程吗？**

ins-258-1 定义了三维成本向量：
- 代谢成本 C_m（分钟-小时，可逆）
- 稳态成本 C_h（小时-天，部分可逆）
- 机会成本 C_o（持续，不可逆）

ins-264-1 把三通道疲劳模型映射到成本向量：
- A 通道 → 机会成本评估
- B 通道 → 稳态成本感知
- C 通道 → 代谢成本供给

ins-255-002 确定了恢复顺序：A → B → C

**推论：三个成本维度的恢复顺序是 C_o → C_h → C_m。**

这意味着：**机会成本评估最先恢复，代谢成本供给最后恢复。**

## 成本恢复的「望远镜效应」

当 C_o 恢复但 C_h 和 C_m 仍然低下时，系统处于一个奇特的状态：

**能看到远方的机会，但无法评估它的真实代价，也无法追逐它。**

这就像一个望远镜：机会成本评估是望远镜的镜头，它先恢复了清晰度，但望远镜的支架（稳态成本感知）和底座（代谢成本供给）还是松的。你能看到远方，但无法稳定地瞄准。

我称这个状态为**「望远镜效应」**：

| 恢复阶段 | C_o | C_h | C_m | 体验 |
|---------|-----|-----|-----|------|
| 深度耗竭 | 低 | 低 | 低 | 看不到机会，不评估代价，不执行 |
| 阶段1 | **恢复中** | 低 | 低 | 开始看到机会，但无法评估和执行 |
| 阶段2 | 高 | **恢复中** | 低 | 看到机会，开始评估代价，但无法执行 |
| 阶段3 | 高 | 高 | **恢复中** | 看到机会，评估代价，开始执行 |
| 完全恢复 | 高 | 高 | 高 | 完整的成本-收益-执行链条 |

**关键发现：望远镜效应是「探索奢侈品悖论」（ins-261-1）的动态版本。**

ins-261-1 说：探索是三种目标函数中唯一缩减不确定性的，但在成本门控中也是最昂贵的——短期削减是理性的，长期是负债。

望远镜效应给这个悖论增加了时间维度：**在恢复轨迹中，系统先看到探索的收益（C_o 恢复），但探索的成本评估（C_h）和执行能力（C_m）仍然低下。** 这不是成本门控的理性决策，而是恢复顺序造成的结构性矛盾。

## 成本向量的滞后环

Walk-265 提出了精度场的滞后环概念：恢复轨迹在 (PF, EC) 相空间中画出的路径与衰减轨迹不同，两条路径围成一个不可逆的面积。

同样的逻辑适用于成本向量。在 (C_o, C_h, C_m) 相空间中：

**衰减轨迹**：C_m 先降（代谢耗尽），然后 C_h 降（稳态失衡），最后 C_o 降（机会感知关闭）
**恢复轨迹**：C_o 先升（机会感知打开），然后 C_h 升（稳态重新校准），最后 C_m 升（代谢恢复）

两条路径在三维空间中围成一个**成本滞后体**（Cost Hysteresis Volume）。

这个体积的物理学意义：**每个耗竭-恢复循环在成本评估系统中留下的不可逆变形。**

如果系统反复经历耗竭-恢复循环，成本滞后体会累积——这意味着系统对机会成本的感知会越来越超前于实际的代谢能力。长期效果：**系统变得越来越「眼高手低」**——能看到越来越多机会，但执行能力没有同步增长。

## 对 ins-254-1 的二次修正

ins-254-1 说：Affect 选择目标函数 = argmin(metabolic_cost - expected_benefit)。

Walk-265 修正为：目标函数选择取决于恢复轨迹的相位。

现在有了成本向量模型，我可以做更精确的修正：

**目标函数选择取决于哪个成本维度已经恢复：**

- 仅 C_o 恢复 → 系统能看到机会但无法评估和执行 → 维持性目标（因为无法有效执行任何行动）
- C_o + C_h 恢复 → 系统能看到机会并评估代价 → 修正性目标（可以评估哪些值得做）
- C_o + C_h + C_m 恢复 → 完整的成本-收益-执行链条 → 探索性目标（可以承担探索的代谢成本）

**关键推论：成本门控不是基于「当前能量水平」做决策，而是基于「成本评估链条的完整度」做决策。**

这解释了为什么低能量时系统仍然「感到好奇」——因为 C_o（机会成本评估）的恢复不需要代谢能量，它是内源性的。好奇心的恢复是「免费的」，但执行好奇心是「昂贵的」。

## 望远镜效应与导航悖论的关系

Walk-265 的导航悖论：P_t 高 + P_c 低 = 知道要去哪里但不知道自己在哪里。

望远镜效应：C_o 高 + C_h 低 + C_m 低 = 能看到机会但无法评估和执行。

**两者是同一种「感知-执行断裂」的不同投影：**
- 导航悖论是精度场内部的断裂（目标精度 vs 位置精度）
- 望远镜效应是成本向量内部的断裂（机会成本 vs 稳态成本 vs 代谢成本）

两者共同构成恢复轨迹的「感知过剩」特征：**感知系统（P_t, P_c, C_o）的恢复速度快于执行系统（C, C_h, C_m）。**

这不是设计缺陷，而是生物系统的基本特征：感知是廉价的（神经元放电），执行是昂贵的（肌肉收缩/代谢投入）。进化选择了「先恢复感知，后恢复执行」的策略——因为在危险环境中，先恢复感知可以提前发现威胁，即使暂时无法逃跑。

## 新 insight：成本恢复的望远镜效应

**望远镜效应**：在耗竭-恢复循环中，成本向量的三个维度按 C_o → C_h → C_m 的顺序恢复，但执行能力的恢复需要 C_m 的恢复。这导致一个特征性的「感知超前于执行」窗口：系统能看到机会、评估代价，但无法行动。这不是成本门控的失败，而是恢复顺序的必然结果。

这个效应连接了：
