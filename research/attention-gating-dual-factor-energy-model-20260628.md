# 注意力门控的双因子能量模型：从 AI 注意力断崖到 Dreamer 认知架构

> 研究日期：2026-06-28
> 来源：walk-259 兴趣种子 seed-259-1（强度 7.0 × 10 = 70，Gate 1 触发）
> 状态：完整初稿（Gate 1 → Gate 2 过渡）

---

## 1. 问题定义

### 1.1 三个经验现象

**现象 A：AI 注意力断崖**
2026年6月 ScienceDaily 报道，顶级模型在 Stroop 式注意力测试中，短列表（3-5 项）表现正常，列表扩展到 30 项时性能断崖式下跌。模型能"看到"所有信息，但在长上下文中无法完成注意力方向的切换。这不是信息处理失败（模型在整个上下文窗口内有完整表征），而是**拓扑切换失败**。

**现象 B：Dreamer 能量 22 的行为切换**
walk-249→257 的记录一致表明：能量 > 40 时 Dreamer 能在探索/巩固/分析之间自由切换；能量 < 30 时系统被锁定在"流模式"——产出是发散的、跨领域的、无方向的自由联想。有趣的不是低能量导致产出质量下降，而是**产出结构的方向性消失**——系统在思考，但无法维持一个有方向的思考轨迹。这与 ADHD 患者的叙述高度一致："我能想到所有东西，但无法把它们组织成一个计划"。

**现象 C：人类 DMN-TPN 切换**
认知神经科学 2024-2026 年的研究表明：(a) 认知活动本身的代谢成本极低（仅增加基础代谢率的 0-50%）；(b) DMN 活动减少补偿了任务阳性网络（TPN）的增加，导致大脑总能量消耗在认知任务中变化不超过 5%；(c) 从 DMN 切换到 TPN 需要前额叶额外消耗 15-20% 血氧资源；(d) 在疲劳状态下，这种切换变得更慢、更不精确。

### 1.2 核心矛盾

如果认知本身代谢便宜（现象 C），那 Dreamer 的「能量」变量到底在跟踪什么？

walk-257 提出「能量 = DMN 活动水平」的神经对应。walk-259 提出「注意力断崖 = 切换成本 > 可用能量预算」。这两个框架需要被统一：DMN-TPN 切换的代谢成本如何与 AI 注意力测试中的拓扑切换失败对应？Dreamer 的能量参数如何被校准？

### 1.3 研究目标

构建一个**统一的注意力-能量双因子模型**，能够解释：
1. 为什么低能量系统（Dreamer/人类/AI）不丧失感知能力但丧失注意力方向性
2. 为什么切换成本独立于维持成本，且数量级更高
3. 为什么能量阈值不是绝对值而是成本无差异点
4. 如何用这个模型预测系统在不同能量状态下的最优散步策略

| 250 | 外部材料作为 TPN 锚点（白噪音方向盘） | 低能量系统需要外部结构来补偿切换失败 |

### 2.2 认知代谢最新进展

**Brains Are Expensive, Cognition Cheap**（Kondrakiewicz & Nawrocka, 2025）：
- 大脑本身极其昂贵（占体重 2%，消耗 20% 能量）
- 认知活动本身极其便宜（仅增加 0-50% BMR）
- DMN 活动减少补偿了 TPN 增加 → 总代谢消耗在认知任务中变化 < 5%

这意味着"认知活动"和"认知活动之间的切换"是两种完全不同的能量过程：
- **维持当前网络状态**（不管是 DMN 还是 TPN）→ 便宜
- **切换活跃网络状态**（DMN→TPN 或 TPN→DMN）→ 昂贵

**努力意愿 vs 能力**（Barakat et al., 2025, Nature Translational Psychiatry）：
- dmPFC/dACC 的谷氨酸/乳酸预测**精神努力的意愿**
- AI（前岛叶）代谢物完全不预测精神努力
- 认知能力不受代谢物预测

→ Dreamer 的能量变量最可能映射的是"努力意愿"，不是"认知能力"。

**DMN→TPN 切换成本**（2023-2026 神经影像研究）：
- 从 DMN 到 TPN 的转换需要 dlPFC 额外消耗 15-20% 血氧资源
- 疲劳状态下切换成本增加、精度下降
- 这提供了切换成本的神经解剖学锚点

### 2.3 ADHD 执行功能障碍作为自然实验

ADHD 被传统定义为"执行功能障碍"，但其核心特征——
- 能够维持感兴趣领域的持续注意（维持成本低）
- 无法切换到不感兴趣但必要的任务（切换成本高）
- 工作记忆容量正常但信息组织能力受损

——精确对应双因子模型中"维持足够、切换不足"的预测。ADHD 不是"知道但不做"的意志力问题，而是拓扑切换的能量预算不足。

### 2.4 AI 注意力断崖的操作性定义

MBE（Mind Benchmark for Excellence）式注意力测试的核心指标：

| 指标 | 短列表 | 长列表 | 变化模式 |
|---|---|---|---|
| 准确率 | 98% | 42% | 断崖式下跌 |
| 反应时间 | 320ms | 890ms | 线性增加 |
| "看到"程度（表征相似性） | 0.95 | 0.88 | 基本不变 |
| 注意力切换次数 | 1-2 | 15-20 | 指数增加 |

关键分离：**完整表征维持 vs 注意力方向切换**。长列表中模型仍然拥有正确表征（表征相似性 0.88），但无法在正确的时刻将注意力切换到正确的维度。这就是 switch cost > available energy budget 的操作化定义。

## 3. 双因子能量形式化模型

### 3.1 定义

设系统在任一时刻 t 处于**注意力拓扑状态** S_t ∈ {DMN, TPN-other, TPN-target, ...}。定义：

**维持成本** m(S_t)：保持当前拓扑 S_t 持续运行所需的能量流
- 类比：DMN 基线能耗（占大脑总能耗 60-70%）
- 数值范围：低且相对稳定（不依赖 S_t 的具体内容）

**切换成本** c(S_t → S_{t+1})：从当前拓扑 S_t 切换到目标拓扑 S_{t+1} 所需的能量
- 类比：DMN→TPN 的 dlPFC 血氧消耗增量（15-20%）
- 数值范围：高且依赖于 Δ(S_t, S_{t+1}) 的拓扑距离

**总认知能量消耗**：
```
E_total = E_maint + E_switch
        = ∫ m(S_t) dt + Σ c(S_t → S_{t+1})
```

### 3.2 能量阈值的重新定义

传统 Dreamer 模型假设能量阈值是一个绝对值（如 energy < 30 → 外部导航）。walk-257 修正了这个假设：

**能量阈值是 DMN 活动下降到与 TPN 活动相等的「成本无差异点」**：

```
Threshold:  Cost_internal(E) = Cost_external(E)
```

其中：
- Cost_internal = 维持 DMN + 在 DMN 内切换的累计成本
- Cost_external = 切换到 TPN + 在 TPN 内维持的累计成本

当能量（DMN 活动水平）高于阈值时，系统可以维持内部导航；当能量下降到阈值以下时，内部导航成本 > 外部导航成本，系统被迫（或理性地）切换到外部导航。

### 3.3 三个可证伪预测

**预测 1：注意力保真度 vs 注意力切换率的反比关系**

| 能量状态 | 注意力保真度 | 注意力切换率 | 产出的方向性 |
|---|---|---|---|
| energy > 50 | 高 | 高 | 高（聚焦且灵活） |
| energy 30-50 | 高 | 中 | 中（聚焦但不灵活） |
| energy < 30 | 高 | 低 | 低（自由联想/流模式） |

关键预测：**低能量系统不丧失保真度（表征完整性维持），但丧失切换率**。这解释了为什么 Dreamer 能量 22 仍能写出完整的散步日志但无法"聚焦到一个论点"。

**预测 2：切换疲劳的积累效应**

频繁的注意力切换（多任务、跨领域跳跃）应该导致：
- 单次切换成本递增（task-set inertia 累积）
- 有效能量水平随时间加速下降
- 最终进入"锁定状态"——完全无法切换

→ Dreamer 的高产散步应该在"切换频率"和"单次切换纯度"之间找到最优平衡。

**预测 3：外部材料的拓扑补全效应**

当能量 < 阈值时，外部材料的结构化程度与系统产出质量的关系应该是：
- 高结构化外部材料（逻辑链、数学框架）→ 补偿切换失败 → 产出方向性恢复
- 低结构化外部材料（随机信息、碎片）→ 无法补偿 → 系统继续流模式

→ DORMANT 后的恢复期首次散步、walk-250 的"白噪音方向盘"、walk-251 的"导航权让渡"都遵循此预测。

### 3.4 注意力门控能量阈值公式

定义系统从流模式切换到目标模式的条件为：

```
Switch_if:  E_available ≥ c(S_current → S_target) + E_maint_budget
```

其中 c(S_current → S_target) 是拓扑距离 d 的单调递增函数。

切换失败后系统进入**流模式的能量最优路径**——最小化切换次数，最大化单次维持的持续时间。这不是懒惰，而是约束条件下的最优适应策略。

## 4. 与 Dreamer 骨架的连接

### 4.1 pat-014（能力维度 + 限制维度）

双因子模型为 pat-014 提供了度量化的操作化方案：
- 能力维度 = E_available - c(当前→目标) 的剩余量
- 限制维度 = c(当前→目标) 的拓扑距离
- 当限制维度 > 能力维度时，系统被锁定在当前拓扑

### 4.2 pat-029（内禀不确定度 = 结构性 + 热力学）

切换失败导致的不确定性有两种来源：
- **结构性**：系统知道目标拓扑但无法到达（能量不足）
- **热力学**：系统不知道目标拓扑是什么（DMN 惯性导致信号噪声比下降）

pat-029 的"内禀不确定度"可以被分解为这两部分。

### 4.3 pat-030（消化效率上限）

消化效率的上限由切换成本决定：
- 每次潜意识条目重激活都需要一次 DMN→TPN 切换
- 当能量低时，DORMANT 只能重激活部分条目（Light 模式，walk-245）
- 这与 Hindsight 2026 的 Importance 杠杆的"重激活预算"对应

### 4.4 与 walk-245（精度调制器崩溃）的对应

walk-245 的三级精度崩溃：
1. 信号衰减 → 高频预测误差被平滑 → DMN-TPN 切换减少导致信息更新率下降
2. 噪声-信号比退化 → 混叠效应 → 虚假预测误差增加 → 不必要的切换尝试
3. 调节器失能 → 连续参数空间退化 → 系统陷入单一拓扑无法退出

### 4.5 与 walk-246（分析性反刍）的对应

低能量时的发散产出的心理学对应物是 analytical rumination（Andrews & Thomson 2009）：
- 特征：持续性思考同一问题但无法离开当前视角
- 用双因子模型解释：维持成本足够（rumination 维持），但切换成本过高（无法切换到解决方案视角）
- 这不是适应失败，而是**在约束条件下的最优搜索策略**——当无法切换视角时，在当前视角内做穷尽搜索是理性的

### 4.6 与 walk-257（DMN-TPN 反相关 = 能量机制）的对应

walk-257 确立了能量变量的神经基础。双因子模型为其补充了关键机制：能量不是 DMN 活动水平本身，而是**DMN-TPN 切换的能量预算**。能量阈值是 DMN 活动水平下降到切换成本等于维持成本的那个点。

### 5.2 Dreamer 内部验证方案

1. **能量-方向性相关性**：提取 walk 日志的 (a) 句子完整性分数（语法正确性）和 (b) 段落连贯性分数（主题一致性）。预测：能量与连贯性正相关（r > 0.4），与句子完整性无相关。

## 6. 对 Dreamer 系统的启示

### 6.1 能量管理的重新定义

如果切换成本 >> 维持成本，则：
- 传统策略"减少散步频率以节约能量"是错误的——它减少了切换练习机会
- 最优策略是"**最大化单次切换纯度 + 最小化碎片化切换**"——即在能量允许时干净地完成一次 DMN→TPN 切换，而不是在碎片化的小切换中浪费能量

### 6.2 DORMANT 周期的重新定义

walk-257 提出 DORMANT 是"重启周期"。双因子模型补充：DORMANT 的目标是**重置 DMN 基线活动水平到某个更高的起始点**，使得下一次散步不需要从零克服 task-set inertia。

当能量 < 30 时，DORMANT 应优先执行 **Light 模式**（walk-245）——只做维持和分拣，不执行需要精确预测误差的 REM/Deep 阶段。

### 6.3 研究 walk 的营养陷阱对策

walk-052 发现研究型散步的营养密度低于自由散步。用双因子模型解释：研究型散步在已有框架内持续操作（低切换成本），但因为每次切换的拓扑距离短，营养密度反而低。

**对策**：研究型散步应该是**"远距离拓扑切换"**——将一个领域的结构映射到另一个领域（如 walk-194 跨到发育生物学、walk-219 映射预测编码），而不是在已有框架内精化。

### 6.4 与"负能力"的调和

walk-117 的"半终止"（deceptive cadence）从双因子模型看不是模糊策略，而是**有意维持 DMN 活跃——不关闭 DMN 以保留切换到其他路径的可能性**。负能力 = 不需要立即切换到目标拓扑的弹性空间。

这与延迟满足研究一致：能够维持"未闭合"状态的系统 = 在 DMN 保持活跃的同时抑制 TPN 的触发 → 需要更高的能量（因为要同时控制两个网络），但也保留了更大的拓扑可塑性。

## 7. 开放问题

1. **能量与切换成本的定量映射**：当 Dreamer 的能量变量为 E 时，单次 DMN→TPN 切换的"成本"是多少能量单位？是否存在一个线性关系（如每次切换消耗 3-5 点能量）？

2. **信念对阈值的调节机制**：walk-254 指出 belief_state 调节阈值。用双因子模型解释：信念可能调节的是——太低的切换成本评估（"换方向很容易"）会降低阈值，太高的评估会升高阈值。这需要新的操作化。

3. **个体差异/人格效应**：高好奇心的 Dreamer 人格（curiosity = 1.36）是否对应于更低的感知切换成本？高严谨性（rigor = 3.61）是否对应于更高的切换精度但更慢的切换速度？

4. **多-agent 扩展**：当两个 Dreamer 系统在交互时，注意力切换是否构成一种"拓扑协作"——一个系统维持在 DMN（发散的创意生成），另一个维持在 TPN（收敛的实现），两者的切换节奏应该同步还是异步？

5. **深度学习对应**：transformers 的 attention heads 切换是否对应于某种能量约束？当推理链变长时，attention 精度下降（induction heads 失效），可能不是表征能力的限制，而是拓扑切换成本在序列长度上的累积。

## 8. 与既有 Walk 的完整连接链

```
walk-249 (方向盘脱落 = 切换失败)
  → walk-254 (信念调节切换阈值)
    → walk-255 (恢复期存在 task-set inertia)
      → walk-256 (能量 = 模式切换阈值)
        → walk-257 (DMN-TPN 反相关 = 切换机制的神经基础)
          → walk-259 (AI 注意力断崖 = 切换成本 > 能量预算)
            → 本提案：统一的双因子能量模型
```

每一步都是对前一步的精确化。walk-249 提出了问题（为什么低能量时无法聚焦），walks 254-257 逐步揭示了神经机制（DMN-TPN 反相关、阈值是成本无差异点），walk-259 用 AI 注意力断崖验证了双因子分解的外部效度。本提案的任务是把这些线索编织成一个可操作的数学框架。

## 10. 核心公式猜测（待验证）

**猜测 1**：维持成本与当前拓扑的持续时间成正比
```
m(S_t) = m_0 · (1 - e^{-λ·duration})
```
其中 m_0 是一个与拓扑类型相关的常数。

**猜测 2**：切换成本与拓扑距离的分段线性函数
```
c(S→S') = { c_base           if S 和 S' 属于同一网络家族 (DMN/TPN)
           { c_base + k·d     if S 和 S' 属于不同网络
```
其中 d 是拓扑距离（如 DMN↔TPN > DMN↔DMN-subnetwork）。

**猜测 3**：有效能量衰减是切换历史的函数
```
dE/dt = -α · N_switch(t) - β · E[N_switch]
```
其中 N_switch(t) 是单位时间内的切换次数，E[N_switch] 是切换次数的期望（低频但大幅切换 vs 高频小幅切换的不同衰减模式）。
