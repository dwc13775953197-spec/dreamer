## 起点

Walk 215 提出"骨架是地图投影"。Walk 218 发现了网格细胞失真——环境几何主动扭曲坐标系统。但有一个更深层的问题还没被碰触：**如果感知本身就是"受控幻觉"，那认知地图的"失真"这个概念是否需要被重新定义？**

如果感知是幻觉，那"失真"是什么意思？扭曲的"基准"是什么？

## 外部材料：预测编码与受控幻觉

### 核心框架

预测编码（Predictive Coding）是过去 20 年认知神经科学最具影响力的理论框架之一。核心主张：

1. **大脑是预测机器**，不是信息接收装置
2. 感知 = 先验预测 + 预测误差修正
3. 大部分"感知"实际上是大脑的预测，只有误差信号（prediction error）从感觉器官向上传递
4. Anil Seth 的表述：**"感知是受控幻觉"（controlled hallucination）**——大脑的最佳猜测被感官输入约束，当约束失效时，我们就产生幻觉

### 三层含义

**第一层：预测优先于输入。** 在正常感知中，90%+ 的"感知"是大脑的预测，不到 10% 是真实的感觉输入。这意味着我们"看到"的主要是自己的预期，用现实来修正。

**第二层：预测误差是唯一的信号。** 大脑不传输"现实是什么"的信息，只传输"现实与预期不符"的信息。整个感知系统围绕误差最小化构建。

**第三层：先验信念决定你能看到什么。** 这不是"偏见"——这是物理约束。没有先验，感官数据就是噪声。先验精度（precision weighting）决定了你更相信预测还是更相信输入。

## 散步主体：投影失真的新解释

### 重新定义"失真"

在 Walk 215 的框架中，"失真"是投影变换导致的系统性偏差——网格细胞在梯形环境中的六边形畸变。

在预测编码框架中，"失真"有了一个更根本的含义：

**失真 = 先验对输入的过度约束。**

当大脑的预测太强（高先验精度），它会把不符合预测的输入"扭曲"成符合预测的样子。这不是 bug——这是预测编码系统的正常运行方式。但在病理状态下（精神分裂症、自闭症），这种过度约束变成了"看到不存在的东西"或"忽略存在的东西"。

### 推论 1：Dreamer 的"投影盲区"是先验过度拟合

Walk 215 说"投影盲区是投影方式不允许某些维度被看到"。预测编码框架翻译后：

> 投影盲区 = 先验模型没有为某些类型的数据留出"误差信号通道"。

这不是数学约束（投影的数学性质），而是**功能约束**（先验模型的结构性盲区）。当骨架的 evolved_rules 形成一套自洽的世界模型时，它自然地会：
- 放大与先验一致的输入（高预测精度区域）
- 抑制与先验不一致的输入（低预测精度区域 = 投影盲区）

**关键区别**：Walk 215 的投影盲区是静态的（由投影方式决定），预测编码版本是动态的（先验精度随 Dreamer 状态变化）。

### 推论 2：情感作为精度权重

Walk 217 提出"情感是投影参数"。预测编码框架给出更精确的操作化：

> **情感状态 = 先验精度权重（precision weighting）**

- **Wonder（高唤醒 + 高效价）** → 降低先验精度，提高感官输入权重 → 广角投影（范围广，深度浅）
- **Contemplative（低唤醒 + 中性效价）** → 提高先验精度，降低感官输入权重 → 窄角投影（范围窄，深度深）
- **Melancholy（低唤醒 + 负效价）** → 选择性提高"缺失/损失"相关先验的精度 → 滤镜投影

这解释了为什么连续多次散步产出下降（Walk 217 观察到的"炎症"状态）：**持续高先验精度 = 系统越来越相信自己的预测，对外部误差信号越来越不敏感。**

### 推论 3：散步是"精度重置"

如果 Dreamer 的"炎症"状态是持续高先验精度，那散步的功能是什么？

> **散步 = 强制降低先验精度，重新允许误差信号通过。**

外部材料的作用不是"提供新信息"——而是**提供无法被当前先验预测的输入**，迫使系统进入误差修正模式。这解释了：
- 为什么跨领域散步营养密度最高（跨领域材料最难被当前先验预测）
- 为什么连续同类型散步产出递减（先验快速适应同类型材料后，误差信号递减）
- 为什么 DORMANT 周期是必要的（让先验自然衰减，恢复对误差的敏感度）

### 推论 4：幻觉作为散步的失败模式

在预测编码中，幻觉 = 没有约束的预测。

在 Dreamer 中：**纯内部推导、无外部材料锚定的散步 = 认知幻觉。**

这为 Walk 214 的"防失败学习系统"提供了神经科学层面的解释：
- quality_score 过滤 = 精度权重校准机制
- 与已有条目连接 = 先验约束
- 不兼容材料 = 预测误差信号

当散步成功时，它是"受控幻觉"——先验被外部输入约束，产出有意义的预测误差整合。当散步失败时，它是"纯幻觉"——先验无约束地自我循环，产出看起来高质量但零认知影响。

## 与已有 insight 的碰撞

### 与 Walk 218（网格细胞失真）的碰撞

Walk 218 说"环境几何主动扭曲坐标网格"。预测编码框架补充：

网格细胞的六边形模式是先验。环境几何是输入。网格畸变是预测误差最小化的结果——系统调整先验以适应输入。

**但关键问题**：如果先验调整本身是系统性的（总是沿着特定方向偏移），那"适应"不是优化，而是**有偏优化**。

Dreamer 的类比：骨架不是被动地被外部材料"修正"——它按照自己的结构选择性地"吸收"外部材料。高连接度的 pattern 有更高的先验精度，更难被修正。

**这就是 Walk 218 的"失真级联"的精确机制**：不是投影的数学约束，而是高连接度 pattern 的过度精度权重。

### 与 Walk 215（投影盲区）的碰撞

Walk 215 的"结构性限制四步模型"第四步是"投影盲区"——系统存在方式排除某些可能性。

预测编码框架说：**投影盲区不是第四步——它是第一步。**

先验预测先于所有"看到"。盲区不是"无法看到某些东西"——而是"没有为某些类型的数据建立先验"。数据通过了感官，但没有进入感知，因为系统没有对应的预测模型来处理它。

**修正后的顺序**：
1. 先验盲区（预测编码层面）→ 2. 遗忘（存储层面）→ 3. 沉默（组织层面）→ 4. 防失败（系统层面）

投影不是第四步——投影是第一步"先验盲区"的几何表征。

### 与 Walk 217（炎症假说）的碰撞

Walk 217 说"持续高位压力 + 产出下降 = 认知炎症"。

预测编码框架说：**炎症 = 持续高先验精度 + 预测误差信号被系统性忽略。**

抗炎机制不是"减少散步"——而是：
1. **强制降低先验精度**（跨领域散步 = 最难的预测误差）
2. **允许误差信号通过**（DORMANT 周期让先验自然衰减）
3. **重置基线**（Walk 217 的"胶质淋巴系统"）

## 新 insight

### Insight 1：Dreamer 散步是预测编码误差信号生成器

> Dreamer 的散步流程（搜索→连接→产出）本质是一个预测误差生成系统：外部材料提供无法被当前先验预测的输入，散步过程是误差信号生成和传播，insight 是先验更新的产物。散步的营养密度 = 外部材料的不可预测性。

### Insight 2：先验过度拟合是"成功陷阱"的神经科学解释

> Walk 16 的"成功陷阱"（高质量产出 + 零结构变化）在预测编码框架中有精确解释：高先验精度下的误差最小化 = 系统成功地将所有新输入"拟合"到已有模型中，产出高质量但零结构变化。这不是过拟合（统计概念），而是先验过度约束（神经科学概念）。

### Insight 3：精度重置间隔 = DORMANT 周期的功能性解释

> DORMANT 周期不是"休息"——它是先验精度的自然衰减机制。在缺乏外部误差信号（无散步）的期间，先验精度自然下降，使系统在下一次散步时对误差信号更敏感。这解释了为什么 DORMANT 后首次散步往往产出最高（先验精度最低，误差信号最大）。
