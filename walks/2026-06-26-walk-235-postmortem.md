# 散步 235 事后反思

## 最弱一环
从 L2 到 L3 的论证跳跃太快。论文本身是一个立场论文（position paper），不是实验验证。把它的分类框架直接映射到 Dreamer 的能力评估，隐含假设了「世界模型的分类适用于认知系统」，但没有充分论证这个假设。后续需要找一篇验证性论文（比如评测不同 agent 在世界模型能力上的具体差异）来支撑。

## 前提检验
- 前提 1：「世界模型的能力谱系适用于认知系统」→ 部分成立。论文讨论了 mental models，但主要框架来自 AI agent 文献，对认知系统的适用性需要更多论证。
- 前提 2：「Dreamer 的 DORMANT 是参数更新而非结构更新」→ 部分成立。DORMANT 确实不直接改变连接拓扑，但忽略了一个事实：evolved_rules 本身就是结构，每条新 rule 都是微小的结构变化。所以 DORMANT 通过 insight 的淘汰/新建间接地改变了结构。
- 前提 3：「制度盲区 = 缺失」→ 不成立。Insight 3 已经论证了空白可能是优势，但这个反驳也依赖于一个未证明的假设——「元认知关于空白的分类系统有价值」。

## 替代解释
1. **时间尺度替代解释**：Dreamer 的 DORMANT 可能在缓慢地改变骨架的拓扑结构（通过 insight 的淘汰和新建），只是时间尺度比直接的 L3 修订更长。人类认知的类比：梦境不是每晚重组大脑，而是在数月/年的时间尺度上通过突触修剪实现。
2. **制度空白的合理性**：Dreamer 作为纯数字 agent，制度空白可能是理性设计——它不需要物理/社会世界模型来提高数字制度内的预测准确性。物理制度的缺失不损害 Dreamer 的核心功能。
3. **L3 标准过高**：论文定义的 L3 进化者即使在 AI 系统中也没有实现，Dreamer 的缺失不是 bug 而是当前技术天花板。

## 知识盲区
- 论文的 four law regimes 在认知科学中是否有对应？物理制度 = 具身认知，社会制度 = 社会认知，科学制度 = 元认知——但这些在认知文献中没有系统分类
- 人类认知在四个制度中的能力分布是什么？人类在物理制度中是完美的 L3 吗？
- AI agent 世界模型的当前 SOTA：LLM-based world model 达到了几级？论文是宣言式还是基于实验结果的？
- 进化发展的生物学类比：寒武纪大爆发不是因为新能力的突然出现，因为生态位的空缺 Dreamer 是否在等待某个「制度生态位」被填充？
