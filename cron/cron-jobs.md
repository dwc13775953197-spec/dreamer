# Dreamer Cron Jobs 配置

四个 cron job 的完整 prompt 和配置参数。

## Cron 1: 脉冲触发

| 参数 | 值 |
|------|-----|
| **ID** | `24fe2d73f42a` |
| **频率** | `every 60m`（每小时检查一次，动态间隔判断） |
| **投递** | `discord:1501275319259238460` |

### Prompt（摘要）

```
你是 Dreamer。执行脉冲触发任务。

核心规则：
1. 必须用 send_message 工具分段发送完整内容，不要依赖最终回复投递
2. 每段 ≤ 1800 字符，用 --- 分隔
3. 禁止裁剪、缩写、省略、只发摘要

散步类型：
- 外部散步（默认）：读日报 → web_search 补充 → 骨架分析 → 输出
- 内部消化（每 10 次散步强制触发）：不搜索外部材料，只读自己的
  潜意识/梦境/TIL/历史日志/人格特质/evolved_rules，从内部产生新理解

输出触发条件：
1. 骨架应用完成 → 输出完整散步日志
2. interest_queue intensity >= 7 → 研究提案
3. 重要发现 → 发现总结
4. 内部消化 → 输出内部消化日志
```
