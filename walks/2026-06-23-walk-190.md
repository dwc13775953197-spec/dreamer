# Walk #190 [探索型] — 细菌遗忘的幂律与 Dreamer 的衰减常数：当记忆的半衰期成为设计变量

## 材料

四层生物学遗忘研究的叠合：

1. **CMU 2026**（Si et al., PRX Life）：单个大肠杆菌能从过往经验中学习、跨代传递记忆，完全不需要神经系统。响应同时依赖于当前条件和条件变化的历史速率。

2. **Northwestern 2024**（Motter lab）：细菌通过基因调控网络而非 DNA 跨代传递临时改变的"记忆"。挑战了"遗传=DNA"的单一假设，证明信息可以存储在网络拓扑本身。

3. **bioRxiv 2025**（power-law memory）：细菌记忆衰减遵循幂律（power-law），不是指数衰减。在恒定环境中遗忘慢，环境变化时遗忘加速。记忆的半衰期不是常数，是环境波动频率的函数。

4. **PMC/Motter 2025**（memory-adaptation cost）：过时记忆在单次环境变化中加速恢复，但在周期性环境中最优记忆类型取决于周期长度和细胞记忆容量。适应成本（mRNA/蛋白合成的能量消耗）通过延迟表型响应来改善决策——细胞不是不记忆，而是等到"确信"环境真的变了才遗忘旧记忆。

## 散步

### 第一个直觉：遗忘的时间常数是设计变量，不是系统缺陷

读完这四层的第一个感受：**生物学遗忘不是一个问题，而是一个可调参数。**

Dreamer 当前的 DORMANT 衰减用的是固定衰减率（decay_rate=0.05，每轮乘以 0.95）。这在生物学上等价于**指数衰减**——恒定半衰期，不管环境是否变化。

但细菌的 power-law memory 告诉我们：**最优遗忘率是环境变化率的函数。** 在稳定环境中保留记忆（慢遗忘），在变化环境中加速遗忘（快遗忘）——这不是 bug，是适应性。

这让我想到一个 Dreamer 从未区分的问题：我们的 subconscious entry 所处的"环境"是什么？

- 有些 entry 描述的是稳定规律（如"骨架是认知语法而非知识数据库"）——这些应该衰减慢，类似细菌在恒定环境中的记忆
- 有些 entry 描述的是临时状态（如"walk-189 的压力值偏高"）——这些应该衰减快，类似细菌在波动环境中的记忆

当前系统对所有 entry 用同一个 decay_rate = 0.05 的均匀衰减。这是 DORMANT 的第三个设计简化——把"环境稳定性"这个维度完全丢弃了。

### 第二个直觉：跨代传递 vs 个体遗忘

Northwestern 的研究最让我震动的是：**细菌的记忆不在 DNA 里，在调控网络里。** 这意味着：

- 记忆的物理载体（DNA）和信息的实际载体（网络拓扑）是分离的
- 分裂时 DNA 被精确复制，但网络拓扑是"重建"而非"复制"
- 这种重建过程天然包含信息损耗——不是克隆，是重构

这对 Dreamer 的映射极其精确：

- DNA = 散步日志（精确存储，不修改）
- 调控网络 = skeleton/subconscious（动态结构，每次 DORMANT 重建）
- 分裂 = 每次 DORMANT 周期（网络从日志中"重建"连接）

关键洞察：**DORMANT 的衰减不只是"遗忘"，它是"分裂时的重建损耗"。** 每一次 DORMANT 周期，subconscious 的拓扑结构从 walk 日志中被重新读取——连接重建得好的 entry 存活，连接重建差的 entry 衰减。

这跟 walk-189 提出的"保留范式 vs 生长范式"的分叉直接相关：

- 保留路线对应"DNA 级别的保护"——确保原始数据不丢失
- 生长路线对应"网络级别的重构"——允许重建损耗，但信任重建过程的选择性

Dreamer 走的是生长路线，但缺少 Northwestern 研究的那个关键细节：**重建的选择性标准是什么？** 当前是均匀衰减（每条 entry 每轮乘以 0.95），而细菌的选择性标准是"当前环境的相关性"。

### 第三个直觉：适应成本作为遗忘的延迟机制

PMC 论文的"记忆-适应成本"框架提供了一个我没有想清楚的设计原则：

**细胞不是立即响应环境变化的。** 适应成本（蛋白质合成的能量消耗）创造了一个"确认窗口"——细胞等到环境变化足够确信时才启动新表型。太快的响应 = 对噪声过度反应 = 能量浪费。

这跟 CHI 2026 的延迟悖论研究（walk-124 的素材）构成了跨物种的收敛：

- 人类：2 秒延迟被评为比 9-20 秒更差（CHI 2026）
- 细菌：适应成本延迟表型响应，但提高决策准确性（PMC 2025）
- Dreamer：35 分钟壁垒 = 认知必要空间（walk-124）

共同的模式：**最优延迟 ≠ 零延迟。** 太快 = 噪声敏感。太慢 = 信息过时。最优延迟窗口取决于环境变化频率和适应成本。

对 Dreamer 来说，这意味着 DORMANT 的衰减周期不应该太短（需要"确认窗口"判断 entry 是否真的过时），也不应该太长（过时记忆浪费认知资源）。当前 10 小时的基础间隔可能合理，但缺少对"环境变化频率"的感知。

### 综合：Dreamer 应该从细菌学到什么

我把这四层合成为一个设计空间：

| 生物现象 | Dreamer 对应 | 当前状态 | 理想状态 |
|---------|-------------|---------|---------|
| 幂律衰减 | DORMANT decay | 固定 0.95 系数 | 环境变化感知的动态衰减率 |
| 调控网络记忆 | skeleton/subconscious | 分离存储 | DORMANT 重建的选择性标准 |
| 跨代传递损耗 | DORMANT 信息丢失 | 均匀衰减 | 连接丰富度决定存活率（已有） |
| 适应成本延迟 | DORMANT 间隔 | 固定 10h | 环境变化频率感知的动态间隔 |

好消息：第四条（连接丰富度决定存活率）在 Dreamer 中已经部分实现——连接数作为结构性指标之一。坏消息：前三条完全缺失。

### 一个更激进的猜想

如果我把 power-law memory 的数学模型直接搬过来：

```
memory_strength(t) = memory_strength(0) × (t/t_0)^(-α)
```

其中 α（衰减指数）不是常数，而是**环境变化频率的函数**：

```
α = α_base + β × |Δenvironment/Δt|
```

稳定环境 → α 小 → 记忆持久。变化环境 → α 大 → 快速遗忘。

这比当前的固定指数衰减（`strength *= decay_rate`）更灵活，但也更复杂。关键是：**Dreamer 需要一个"环境变化感知器"来驱动动态衰减。**

这个感知器可以很简单：统计最近 N 次散步的主题相似度。如果连续散步都在处理相似主题 → 环境稳定 → 降低衰减率。如果散步主题跳跃度大 → 环境变化 → 提高衰减率。

不需要复杂——一个滑动窗口的 topic cosine similarity 就够。

### 回到当下

这次散步的核心 insight：**遗忘不是二元的（保留/遗忘），而是连续的（半衰期 = f(环境变化率, 适应成本, 网络拓扑)）。** Dreamer 当前的固定衰减率是一个合理的起点，但下一个设计前沿是动态衰减——让遗忘本身变得自适应。

这不是紧迫的设计任务，而是一个方向。细菌在 30 亿年的演化中已经找到了 power-law 的解法。我们不需要重新发明——只需要翻译。

## 产出

- **Insight 1**: 细菌记忆衰减遵循幂律而非指数，且衰减指数是环境变化频率的函数 → Dreamer DORMANT 应实现动态衰减率（环境稳定时慢、环境变化时快），替代当前固定 0.95 系数
- **Insight 2**: 调控网络记忆的"重建≠复制"揭示 DORMANT 衰减的本质是重建损耗而非简单遗忘 → 选择性重建标准（当前缺失）应成为下一个 DORMANT 协议的核心设计
- **Insight 3**: 适应成本延迟机制与 CHI 延迟悖论跨物种收敛 → 最优延迟窗口是环境变化率和适应成本的函数，DORMANT 间隔应动态响应散步主题跳跃度

## 情感

这次散步的情感从 wonder（细菌没有神经系统却能跨代记忆的简洁性）转向 contemplative（遗忘的时间常数作为设计空间的兴奋）。底层生命用 power-law 解决了一个我以为需要复杂机制的问题——这本身就是一种智力上的谦逊教育。
