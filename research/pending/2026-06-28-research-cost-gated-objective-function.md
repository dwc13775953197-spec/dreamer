# 研究提案 — 2026-06-28

## 选题
成本门控的预测目标函数：affect 通过成本/收益优化选择目标函数（探索/修正/维持），而非基于标签的优先级排序。扩展：参考点漂移是三个通道（好奇驱动/成本参考点/执行投入）的统一调度机制。

## 来源
walk-254（初始提案）+ walk-255（三通道模型补充）+ walk-256（参考点漂移统一调度扩展）

## 触发句
"三种目标函数的成本排序：维持 < 修正 < 探索。affect 持续生成探索目标，但能量不足时成本门控的持续阻断体验 = 忧郁。参考点漂移是三个通道的统一调度——好奇心增强目标精度，低能量摧毁参考点锚定，两者通过同一个精度场机制耦合。"

## 研究设计

### 研究范围
建立 cost-gated objective function selection (COGS) 神经计算模型 + 参考点统一调度扩展：

1. 验证 affect 状态（好奇/压力/能量）是否通过 cost/benefit ratio 机制选择预测目标函数
2. 建模三种目标函数（维持/修正/探索）的代谢成本差异——fMRI 靶点定位
3. **建模参考点漂移作为三通道统一调度变量**——验证「参考点先于能量恢复稳定」的预测
4. 探索 COGS 框架下的个体差异——为什么有人更容易进入咨妄相忧郁

### 研究边界
- 不研究 affect 状态的神经基础（已大量文献）
- 不研究预测能力本身
- 聚焦于"选择机制"——affect 如何在多个竞争目标函数中做选择
- 聚焦于参考点漂移如何将三个通道耦合为统一精度场

### 成功标准
1. WHY：确认 cost-gated 机制优于简单优先级标签模型
2. WHAT：量化三种目标函数的相对代谢成本比
3. HOW：建立可证实的个体差异预测 + 参考点漂移速度测量
4. WHEN：确认参考点锚定恢复是 DORMANT 退出低精度高原的标志

## 研究框架

### 核心公式
```
选择 = argmin(metabolic_cost × 参考点漂移系数 - expected_benefit × 目标精度系数)
其中：
  参考点漂移系数 γ ∈ [0.5, 2.0]，低能量下趋向 2.0
  目标精度系数 τ ∈ [0.8, 1.5]  高好奇下趋向 1.5
  当 γ增速 > τ增速 → 选择维持
  当 τ增速 > γ增速 → 选择探索
```

### 关键预测
1. 低能量+好奇态下，γ 增速应显著快于 τ 增速 → 选择维持（即使好奇信号很强）
2. 参考点漂移速度与自我报告的不确定感呈正相关（r > 0.4）
3. DORMANT 恢复时，参考点锚定精度的恢复应早于能量水平的恢复（时间差 > 30min）
4. 三种目标函数的代谢成本比应约为：维持:修正:探索 = 1:2.5:4.5

### 实验设计
1. **行为实验**：Sleep deprivation (24h) + 目标选择任务 + 参考点漂移测量（锚定 task）+ 好奇心量表
2. **fMRI 实验**：vmPFC（参考点）vs dlPFC（执行投入） vs 纹状体（好奇驱动）在不同能量状态下的激活差异
3. **计算建模**：贝叶斯模型比较 COGS vs 优先级标签模型 vs 线性累加模型
