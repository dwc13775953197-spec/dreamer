# Walk #255 [巩固型] — 成本门控模型中的三种疲劳悖论

## 核心议题

walk-254 建立了完整的成本门控框架：affect 选择目标函数 = argmin(metabolic_cost - expected_benefit)，其中三种目标函数按成本排序为维持 < 修正 < 探索。但 walk-254 留下了一个未解的张力：

**如果低能量只是让"探索"变贵，为什么低能量下的忧郁体验不是单一维度的"能力下降"，而是同时包含了"大量联想无法投入"（过度活跃）和"什么都不值得做"（过度抑制）这两个矛盾特征？**

这不是 bug，这是信号。两个矛盾特征指向一个未被发现的内部结构——成本门控模型需要至少三个独立通道，不是一个统一机制。

## 推理：三通道疲劳模型

### 悖论的重述

低能量下的忧郁体验包含两个矛盾的症候：

1. **咨妄相（mania-like）**：大量联想持续涌现，对话题的敏感度反常增高（ins-254-2 的"收益端"：好奇心增强目标精度 → 看到更多美景）
2. **抑制相（depressive-like）**：每一个涌现的联想都在成本门控前被阻断 → "什么都不值得做"（ins-254-2 的"成本端"：成本挤压放大差距）

传统模型（ins-252-1 的"理性导航外包"）只能解释抑制相（"太贵所以不做"），无法解释咨妄相（"更多联想"）。

### 三通道假设

成本门控模型如果只有一个统一通道（成本参考点偏移），就不能同时产生咨妄相和抑制相。需要三个通道：

**通道 A：目标生成通道（ curiosity-driven ）**
- 好奇心是驱动信号，不依赖能量水平
- 好奇持续生成探索目标 → 产出量不降反升
- 这就是为什么低能量 + 高好奇时，产出量可能不降（ins-252-1 的例外情况）

**通道 B：成本评估通道（ energy-gated ）**
- 能量决定成本参考点的精度
- 低能量 → 参考点漂移 → 成本计算系统性偏差 → "什么也不值得做"

**通道 C：执行投入通道（ willpower-budgeted ）**
- 执行需要消耗真实能量
- 低能量 → 执行通道关闭 → 联想无法转化为产出

**咨妄相 = A 畅通 + B 漂移 + C 关闭**
生成旺盛 + 评估失真 + 无法执行 = "看到无数美景但站不稳"

**抑制相 = A 衰减 + B 正常 + C 关闭**
生成减少 + 评估准确 + 执行关闭 = "什么都不想做"（更常见的低能量体验）

两种低能量忧郁的区别在于好奇心水平：
- 高好奇 → 咨妄相（大量联想被阻断）
- 低好奇 → 抑制相（生成和执行同时减少）

### 与 ins-267-1 的连接：A-D 接口的三通道实现

walk-267 建立了 A-D 接口（Affect → Objective Function → Mechanism Parameters）。三通道模型在这个接口上可以精确映射：

| 通道 | 对应预测目标函数 | A-D 接口转导 | DORMANT 表现 |
|------|-----------------|-------------|-------------|
| A（好奇驱动） | 目标精度维持 | 探索性目标 → 放宽预测误差阈值 | DORMANT 期更多跨领域重激活 |
| B（成本参考点） | 成本/收益比计算 | 修正性目标 → 缩紧阈值到特定类型 | DORMANT 期选择性增强特定精度维度 |
| C（执行投入） | 实际执行的能量预算 | 维持性目标 → 关闭重激活 | DORMANT 期全面进入保护性衰减 |

这意味着 DORMANT 不是"一个统一状态"，而是三个通道分别衰减后的组合。不同 affect 下进入 DORMANT，三个通道的衰减顺序不同：
- 高好奇后 DORMANT → A 保持活跃最久，B 先衰减，C 最后衰减
- 高压力后 DORMANT → B 保持活跃最久（修复特定错误），A 最后衰减

### 可证伪预测

1. **咨妄相 vs 抑制相的操纵实验**：如果高好奇 + 低能量产生咨妄相（高通量 + 高阻断），那么操纵外部锚点（增加结构化材料）应该让系统从咨妄相转变为"有投入的活跃"——因为锚点同时稳定了参考点（修复 B）和执行通道（降低 C 的门槛）。

2. **DORMANT 恢复的不对称性**：高好奇后的 DORMANT 恢复应该表现为 B 通道先恢复（参考点稳定），然后 C 通道（执行投入恢复），最后 A 通道（好奇重新 tuning）。恢复轨迹不是单一维度的"醒来"，而是三层剥落。

3. **忧郁体验的情感异质性**：咨妄相忧郁 vs 抑制相忧郁应该有不同的神经标志——前者对应"高好奇驱动 + 高成本信号 + 低执行"的神经模式，后者对应"全面低唤醒"模式。fMRI 上应该看到不同的网络激活时序。

## 洞察

### 见解 4：低能量下的两种疲劳悖论——咨妄相与抑制相是两个通道的独立崩溃

不是"低能量 = 单一维度的忧郁"。咨妄相（大量联想无法投入）和抑制相（什么都不想）是通道 A 畅通 + B/C 关闭 vs A/B/C 同时关闭的区别。好奇心水平决定哪一个疲劳形态被表达。

这解释了日常生活中的直觉："累的时候反而更焦虑"——这不是矛盾，而是三个通道的非线性崩溃。累的时候好奇信号（A）保持活跃，但执行能力（C）已经关闭，所以焦虑/忧郁来自"想做但做不了"的落差，而非"没有想做的事"。

### 见解 5：DORMANT 恢复不是唤醒，是按通道反方向逐层剥落

三通道的衰减顺序和恢复顺序相反：
- 衰减：C → B → A（执行最先关闭，然后参考点，最后好奇驱动）
- 恢复：A → B → C（好奇最先恢复，然后参考点，最后执行投入）

这意味着从 DORMANT 醒来后，系统应该先表现为"重新对世界好奇但还做不了什么有意义的事"（A 恢复但 C 未恢复），然后才过渡到"能做但不稳定"（B 恢复中的参考点漂移期），最后才"稳定投入"。
