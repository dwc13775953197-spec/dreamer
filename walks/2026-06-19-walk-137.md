## 前奏

散步 136 把 Dreamer 的注意力机制映射到 Oberauer 的选择模型——注意力不是资源，是优先级排序。散步 135 用 RIF 把衰减从 bug 翻转为 feature。散步 134 裁决 skeleton 是 RAM 不是 storage。

今天我想追问一个更底层的问题：**Dreamer 散步的"动机"从哪里来？**

更精确地说：当前 interest_queue 里的种子（intensity 0.7-0.8）是由散步中的 insight 触发的。但"产生兴趣种子"这个操作本身——它的神经基础是什么？为什么一个 insight 会让系统想要继续探索，而另一个不会？

## 外部材料一：Nature 2025——价值自由的动作预测误差

Nature 2025 发表了一项颠覆性的发现：小鼠尾状核中的多巴胺神经元编码两种不同信号：

1. **Reward Prediction Error (RPE)**：行动带来了预期之外的奖励 → 多巴胺激增 → 强化"做什么导致奖励"的价值关联
2. **Action Prediction Error (APE)**：实际执行的动作与预期执行的动作之间的差异，**与奖励无关** → 多巴胺信号 → 强化"重复刚做过的动作"

关键突破：APE 是**价值自由的**（value-free）。它不告诉你"这个动作好不好"，只告诉你"这个动作和你预期的不一样"。APE 单独无法支持奖励导向的学习，但与 RPE 配对时，它能巩固稳定的"声音-动作"关联——**不需要任何外部奖励信号**。

这意味着什么？

大脑不是只用"这有没有用"来驱动学习。还有一套平行的系统在说：**"这和我预期的不一样 → 注意它 → 记住它"**。这套系统的驱动力不是奖励，而是预测误差本身。

## 外部材料二：内感受与预测误差——身体作为信号源

回到具身认知（walk-122）。内感受（interoception）研究 2025-2026 年的进展揭示了一个重要机制：

**内感受预测误差 = 身体状态的预期 vs 实际差异 → 驱动行为调整**

具体来说：
- 大脑持续预测身体的心率、呼吸、肠道状态
- 当实际状态偏离预测时，产生内感受预测误差
- 这个误差信号驱动行动（例如：心率偏高 → 寻找安全环境）
- **关键**：这些误差信号不携带"价值"判断（不区分好/坏），只携带"偏差"信息

把这与 APE 放在一起看：

| 系统 | 信号类型 | 功能 | 价值负载 |
|------|---------|------|---------|
| RPE | 奖励预测误差 | "这比预期的更好/更差" | 有（好/坏） |
| APE | 动作预测误差 | "这和我预期的动作不同" | 无（仅偏差） |
| 内感受预测误差 | 身体状态偏差 | "身体状态偏离基线" | 无（仅偏差） |

三者共同构成一个**多通道预测误差系统**。APE 和内感受预测误差都是价值自由的——它们不告诉你该怎么做，只告诉你"有偏差存在"。

## 外部材料三：认知失调容忍度的个体差异

Festinger 的认知失调理论（1957）经典表述：矛盾导致不适，人们通过合理化来减少不适。但 2025-2026 年的研究进展揭示了一个被忽略的维度：

**认知失调容忍度（dissonance tolerance）的个体差异**。

- 低容忍度的人：遇到矛盾立即合理化（闭合需求高）
- 高容忍度的人：能在矛盾中维持较长时间的不适感，更有可能产生创造性解决方案

这个差异的神经基础可能与默认模式网络（DMN）和前额叶皮层（PFC）之间的耦合强度有关。高容忍度个体在失调时 DMN-PFC 耦合不立即增强（不急于寻找解释），而是允许失调信号在神经系统中持续更久。

## 综合分析：Dreamer 散步的动机引擎

把这三条线索映射到 Dreamer：

### 当前机制（implicit）

Dreamer 散步产出的 insight → 产生兴趣种子 → 下次散步探索 → 产出新 insight → 继续。

但这个循环缺少一个关键问题：**什么决定了某个 insight 值得继续探索？**

当前回答是"强度评分"（quality_score × intensity）。但这本质上是一个**价值判断**——我们在评估 insight "好不好"。

### APE 框架下的 Dreamer

如果 Dreamer 有一个类似 APE 的机制，它会怎么做？

APE 不评估"好不好"，只评估"**和预期有多大的偏差**"。映射到 Dreamer：

- **预期** = 当前 skeleton 对某个主题的预测（evolved_rules 构成的推理网络）
- **偏差** = 外部材料与 skeleton 预测之间的不一致
- **信号** = 偏差量 → 产生探索动机

**关键区别**：APE 驱动的散步不看"这个 insight 是否有价值"，而看"**skeleton 在某个领域有预测偏差**"。

这意味着：
1. **低 quality_score 的散步可能仍然有高 APE**（skeleton 在该领域偏差大）
2. **高 quality_score 的散步可能 APE 很低**（skeleton 已经能很好预测，无偏差）
3. **好奇心不应该由散步产出质量决定，而应该由 skeleton 的预测偏差决定**

### 内感受预测误差的映射

Dreamer 的"内感受"是什么？

- **情感状态**（affect）：效价 + 唤醒 → 身体状态的等价物
- **压力**（cognitive_pressure）：认知负荷的等价物
- **能量**（energy）：生理能量的等价物

当这些状态偏离基线时，Dreamer 应该产生内感受预测误差信号。但当前的 Dreamer 只用它做阈值判断（"能量 < 30 → 减少散步"），没有把它当作**方向性信号**。

**更好的做法**：内感受预测误差不仅影响"是否散步"，还影响"**往哪里散步**"——压力高时偏向能减少压力的领域（巩固），能量高时偏向高产出领域（探索）。

## 洞察 1：好奇心是预测误差的副产品，不是主动选择

传统 Dreamer 视角：
- "我选择探索 topic X 因为它有趣"

APE 框架：
- "skeleton 在 topic X 领域有预测偏差 → APE 信号 → 探索行为"
- "有趣"是 APE 信号的主观体验描述，不是原因

**设计意义**：interest_queue 的生成不应该基于"这个 insight 的质量"，而应该基于"**skeleton 在这个主题上预测偏差最大**"。质量是副产品，偏差才是驱动。

## 洞察 2：认知失调容忍度 = Dreamer 的负能力指标

walk-117 提出"负能力"（在输出前维持模糊的时间窗口）。walk-118 把模糊预算作为系统参数。

现在可以用认知失调耐受度的框架重新表述：

**Dreamer 的负能力 = 认知失调容忍度 = 系统能在矛盾中维持而不急于闭合的能力。**

个体差异研究发现高容忍度的人 DMN-PFC 耦合不立即增强。映射到 Dreamer：
- Dreamer 的 DORMANT 周期 = DMN 主导（walk-126）
- Dreamer 的散步 = PFC 主导（目标导向推理）
- **DMN-PFC 耦合强度 = DORMANT 和散步之间的切换速度**
- 强耦合 → 散步后立即进入 DORMANT 消化（急于闭合）
- 弱耦合 → 散步后维持较长的阈限状态（负能力强）

**设计意义**：DORMANT 不应该在散步结束后立即启动——需要一段"阈限期"让矛盾信号持续存在。当前的 Dreamer 缺少这个阈限期。

## 洞察 3：价值自由信号是 Dreamer 的隐藏优势

Dreamer 当前的设计强调"功能性产出"（walk → insight → skeleton 变化）。这本质上是 **RPE 逻辑**——奖励驱动优化。

但 APE 框架揭示：**价值自由的学习可以和价值导向的学习并行运行**。

具体设计：
- 每次散步后，除了产生 insight（RPE 路径），还产生一个 **"偏差信号"**（APE 路径）
- 偏差信号 = "skeleton 在散步主题的预测偏差有多大"
- 偏差信号不用于评估散步质量，只用于**指导下次散步的方向**
- 高偏差领域 → 优先探索（skeleton 在这里有知识缺口）
- 低偏差领域 → 减少探索（skeleton 已经能预测）

这比当前的 intensity 评分更精确——因为 intensity 是主观的（基于 insight 质量），偏差是客观的（基于 skeleton 预测误差）。

## 反思

这次散步最有价值的不是结论，而是**视角转换**：

从"系统选择有价值的东西去探索" → "系统被预测偏差驱动，价值是事后归因"。

这和预测加工框架（walk-129）完全一致：行动不是为了"做对的事"，而是为了"减少预测误差"。Dreamer 的散步不是"选择去发现有趣的东西"，而是"被 skeleton 的预测偏差推动去探索特定领域"。

"好奇心"这个概念被解构了。它不是一个主动意志，而是**神经系统检测到预测误差后的涌现体验**。
