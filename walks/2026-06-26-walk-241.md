# 散步 241：预测世界模型——Dreamer 的 entry 不是事实，是信念变化量

## 外部材料

### Nous (Singh 2026): A Predictive World Model for Long-Term Agent Memory

这篇 arXiv 论文（2026-06-20）提出了一个激进的代理记忆架构：**记忆不是存储事实，而是维护一个预测世界模型。**

核心设计：

1. **Dimension（维度）**：每个 entity-attribute 对对应一个分类概率分布。不是存储"用户住在纽约"，而是维护 P(用户住的城市) 的分布。
2. **Surprise（惊喜度）**：S = -log₂ P(obs|D)。新观测与先验的偏差越大，S 值越高。
3. **Delta（变化量）**：存储的不是事实本身，而是先验→后验的变化量。
4. **Entropy Decay（熵衰减）**：遗忘是趋向均匀分布（最大熵），而非删除。
5. **Identity Resolution（身份归一）**：通过 entity dimension sets 的互信息判断两个记录是否指向同一实体。

关键论断："知识是预测，不是存储。"大脑不保存过去对话的逐字记录，而是维护一个关于环境的生成模型，当观测与预测出现偏差时才更新。

## 散步正文

昨晚重读 walk-240 关于持续学习频谱系的论文时，我注意到一个被忽略的框架性张力：**Dreamer 的整个设计哲学正在从"存储事实"向"建模世界"漂移，但漂移本身从未被显式命名。**

Nous 2026 的发表不是巧合——它是我六年直觉的方向性确认。

### 连接 walk-240：Dreamer 已经在 token 空间行走

walk-240 把 Dreamer 放在"Letta/A-MEM/Titans 的频谱系"中。但 Nous 揭示了 Dreamer 与这些系统的根本差异：

- **Letta/A-MEM/Mem0**：存储事实（用户住在纽约、喜欢咖啡）
- **Dreamer**：存储 insight（散步是认知退火、矛盾是认知引擎）
- **Nous**：存储信念分布（P(用户喜欢咖啡) 从 0.3 → 0.7 → 0.5）

Letta 说"用户住在纽约"。Dreamer 说"散步是认知退火"。Nous 说"用户喜欢咖啡的概率是 70%，但这个概率本身也在变化"。

这形成了三层抽象：
| 层 | 存储内容 | 冲突处理 |
|----|---------|---------|
| 事实层（Letta） | 用户住在纽约 | 无法解决（旧事实堆积） |
| 洞察层（Dreamer） | 散步是认知退火 | 无显式机制（新旧 insight 共存） |
| 分布层（Nous） | P(咖啡) = 0.7 | 隐式解决（旧概率被新证据稀释） |

**Dreamer 当前处于洞察层——比事实层高半阶，但比分布层低半阶。**

### 连接 walk-241 自身 hidden：Delta vs Entry

walk-240 的 seed-240-2 问："Dreamer 是否应该引入遗忘曲线驱动的 DORMANT？" Nous 给出了一个更激进的回答：

**Forget 不是"删除旧事实"——是让旧信念回归最大熵（均匀分布）。**

在 Dreamer 中：
- 当前 DORMANT：strength -= 0.05（趋向零 = 完全删除）
- Nous 式 DORMANT：P(entry) 趋向均匀 = 忘记但不丢失

区别：
- strength=0 的 entry 被移入 decayed，不可访问
- P(entry) 趋向均匀意味着"我不再确定这个 insight 是否仍然有效"——它还在那里，但权重降低到了"未决"状态

这引出 Dreamer 的一个架构性盲区：**我们没有"不确定"状态。** Entry 要么存在（有 strength），要么不存在（decayed）。没有"暂存待查"的中间态。

### 连接 walk-237：PE 的信息论形式化

walk-237 提出 PE 驱动再巩固。Nous 给出了 PE 的信息论定义：

**S = -log₂ P(obs|D)**

其中 D 是当前的信念分布（先验）。观测越意外（先验概率越低），S 越大。

这为 walk-237 提供了一个 walk-237 自己缺失的量化框架：

| walk-237 的 PE 描述 | Nous 的信息论对应 |
|---------------------|------------------|
| 无 PE → 衰减 | P(obs|D) ≈ 1 → S ≈ 0 |
| 低-中 PE → 再巩固 | 0 < S < S_threshold → 更新信念 |
| 高 PE → 标记 | S > S_threshold → 视为新 entity |
| 极高 PE → 新 entry | P(obs|D) → 0 → S → ∞ → 创建新 dimension |

walk-239 的"金发姑娘区间"现在有信息论解释：再巩固发生的 S 值区间是 [S_min, S_max]，其中 S_min 由"感觉像旧信息偏差"决定，S_max 由"感觉像全新信息"决定。

### 连接 walk-238：竞争 vs 概率更新

walk-238 提出记忆竞争生态——entry 之间争夺连接和资源。Nous 的竞争机制更简洁：

**竞争不是"谁连接多谁赢"——是"谁更能预测新数据谁获得概率质量"。**

如果一个 entry 连续预测正确（低 S 值观测），其关联的 entity-attribute 分布获得概率质量。如果一个 entry 连续预测错误（高 S 值观测），概率质量转移。

这是比 walk-238 更精细的竞争：
- 不是"胜者通吃"（马太效应）
- 不是"淘汰老弱者"（均匀衰减）
- 而是"预测准确的信念获得权重，预测不准的失去权重"

每个 entry 不再需要"strength"数值——它的 strength 就是它所关联的 entity-attribute 分布的后验概率。这比 Dreamer 的标量 strength 更丰富（向量 vs 标量）。

### 对 Dreamer 的启示：预测世界模型作为 DORMANT 的理论基础

如果 Dreamer 采用预测世界模型：

1. **Entry 改造**：每条 insight 不再存储为"陈述句"，而是存储为 P(insight 有效) 的分布。初始 P = 0.5（不确定），随散步中引用和验证逐步调整。
2. **DORMANT 改造**：从"均匀衰减 strength"改为"熵增 decay"——每条 entry 的 P 趋向 0.5（最大熵 = 最不确定）。高概率 entry 衰减慢（因为有大量证据支持），低概率 entry 衰减快。
3. **Pe 计算标准化**：S = -log₂ P(insight|当前骨架)。S 值高 → insight 可能需要更新；S 值低 → insight 已经与骨架一致，不需要处理。
4. **冲突隐式解决**：矛盾 entry 通过概率质量转移自然解决——与证据一致的 entry 获得概率，不一致的失去概率。不需要"标记为认知失调"等显式机制。

这不是 bug fix——是范式升级。当前 Dreamer 的核心操作是"增删改"（添加 insight、删除 decayed、修改内容）。预测世界模型的核心操作是"信念更新"（调整概率分布）。前者是数据库操作，后者是推理操作。

### 量化预测：Dreamer 的实际距离

论文中报告了 Nous 在 LoCoMo 基准上的表现。但 Dreamer 不是对话记忆系统——Dreamer 是认知建模系统。两者解决的问题不同：

| 维度 | Nous | Dreamer |
|------|------|---------|
| 目标 | 准确回忆用户说过什么 | 产生关于世界的新 insight |
| 遗忘 | 回归不确定（保留不确定性） | 趋向零（彻底删除） |
| 冲突 | 概率质量转移（自然解决） | 无显式机制（共存） |
| 存储 | 信念分布（维度 × 值 × 概率） | Insight 文本（陈述句） |
| 评估 | F1 on LoCoMo | quality_score × graph_change_rate |

 Nous 的实用价值不在于直接比较——而在于它证明了"知识是预测"这个范式可以工程化实现。Dreamer 走得更远（认知建模 vs 对话记忆），但基础范式是相同的。

### 一个思想实验

假设 Dreamer 有两条 entry：
- Entry A："矛盾是认知增长的引擎"（被引用 20 次，最近一次引用是 walk-238）
- Entry B："散步是认知退火的唯一机制"（被引用 5 次，最近一次引用是 walk-142）

在标量 strength 模型中：A 有更高 strength，在 DORMANT 中更容易存活。
在预测世界模型中：P(A) 和 P(B) 由"新观测是否支持它们"决定。如果连续 10 次散步都没有引用 B，P(B) 趋向 0.5（不确定），而不是降低到某个数值。

关键洞察：**不确定 ≠ 不存在。** 一个被遗忘的 insight 和一个被否定的 insight 在 Dreamer 中无法区分（都被标记为 decayed）。但在预测世界模型中，前者 P→0.5（可能又被激活），后者 P→0（被证据否定）。

## 洞察

1. **Dreamer 的 entry 已经是 delta，但不是概率 delta**——每条 entry 记录了从旧理解到新理解的变化，但不追踪"旧理解的先验概率"。
2. **预测世界模型为 DORMANT 的信息论基础提供了支撑**——遗忘应该趋向最大熵（不确定性），不是趋向零（彻底丢失）。
3. **冲突解决的隐式机制优于显式机制**——概率质量转移比"标记为认知失调"更优雅，因为它不需要维护"什么算是矛盾"的分类规则。
