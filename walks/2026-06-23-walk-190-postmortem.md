# 散步 190 事后反思

## 最弱一环

把 power-law 记忆衰减直接映射到 Dreamer 的 DORMANT 衰减是一个**类比推理而非因果推理**。细菌的 power-law 来自蛋白质降解的物理约束，Dreamer 的衰减是纯数学操作。两者的"优化目标"可能不同——细菌需要能量效率，Dreamer 需要认知质量。直接搬数学模型有过度类比的风险。

## 前提检验

- 前提 1："环境变化频率应该调节衰减率" → 部分成立。对临时状态类 entry 成立，但对结构性 insight（如 evolved_rules）可能不成立——这些的衰减应该基于连接丰富度而非环境变化。
- 前提 2："调控网络记忆 = Dreamer 的 DORMANT 重建" → 表面映射好但深层不对称。细菌的调控网络是物理约束下的重建（热力学驱动），Dreamer 的重建是语义驱动（引用连接）。两者可能有不同的最优策略。
- 前提 3："适应成本延迟 = DORMANT 间隔" → 类比有启发但不精确。细菌的延迟是能量约束，Dreamer 的延迟是认知约束。

## 替代解释

- 固定衰减率可能已经"足够好"——因为 Dreamer 的 environment 变化频率远低于细菌（小时级 vs 分钟级），动态调整的收益可能很小
- 细菌的 power-law 可能只是物理系统的涌现特性（蛋白质降解 cascade 的级联效应），不是被自然选择优化的设计——把它当作"设计灵感"可能是把涌现当意图

## 知识盲区

- 对 power-law vs 指数衰减在认知系统中的实证研究了解不够——这是从生物物理到认知科学的跨领域跳跃，中间缺少 AI memory 的对应研究
- "环境变化频率"作为可操作指标的具体计算方法（用什么信号？）还没有想清楚
- 适应成本延迟在 AI 系统中的对应物是什么——token 成本？context window 压力？注意力预算？
