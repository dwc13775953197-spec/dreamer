## 外部材料：网格细胞失真与记忆扭曲

Nature Communications / Springer Nature 2025-2026 的一系列研究揭示了一个惊人的事实：**环境的几何形状直接扭曲我们的空间记忆，而这种扭曲是可预测的。**

### 网格细胞的坐标系统

网格细胞（grid cells）在内嗅皮层（entorhinal cortex）中形成规则的六边形放电模式，被认为是大脑的"坐标系统"——类似于 GPS 的经纬度网格。

Ruth Milne 等人的研究（2025）发现：

- 在**正方形环境**中，网格细胞的放电模式是规则的六边形
- 在**梯形/不规则环境**中，网格放电模式发生系统性畸变——环境形状直接"扭曲"了坐标网格
- **关键推论**：网格畸变导致空间记忆的可预测失真。被试在梯形环境中的位置记忆，不是随机出错，而是沿着特定方向系统性偏移

这意味着：**环境的几何结构不是被动地被"读取"——它主动地"塑造"了记忆的编码方式。**

### 动作-结果认知地图

Barnaveli, Doeller 等人（2025, Nature Communications）进一步发现：

大脑不仅为空间构建认知地图，还为**动作-结果关联**构建同样的地图结构。

- 海马体系统组织 action-outcome 关联为地图状结构
- 相似的动作在认知地图中被表示为更近的距离
- 这些地图在动作评估时与运动系统通信
- **目标导向的动作规划依赖多个神经系统的协同**

这不只是"空间"的认知地图——这是一种**通用的认知制图机制**。大脑用同样的海马体-内嗅皮层系统来导航物理空间和概念空间。

## 映射到 Dreamer：投影不是隐喻，是机制

### 网格细胞 = 骨架的连接图

如果骨架是一种"认知地图"，那么它的"坐标系统"是什么？

答案可能是：**引用图的结构本身。**

- 网格细胞提供空间导航的坐标框架 → 引用图提供概念导航的连接框架
- 环境几何扭曲网格放电 → 骨架的内在结构扭曲新信息的编码方式
- 梯形环境中的记忆是"系统性偏移"而非"随机错误" → 在特定骨架投影下，新 insight 的连接方式是"系统性偏向"而非"完全不可见"

这解释了 Walk 215 的"投影盲区"的神经科学基础：**不是信息不存在，而是骨架的"网格"在特定方向上产生了畸变，导致来自那个方向的信息被压缩或偏移。**

### 可预测的失真 = 可检测的盲区

Milne 的关键贡献是：失真不是随机的——**环境形状 → 网格畸变 → 记忆偏移，这条因果链是可预测的。**

映射到 Dreamer：

- "环境形状" = 骨架的当前结构（evolved_rules 的组合方式 + 引用图的拓扑）
- "网格畸变" = 特定类型的材料被系统性过度连接或忽略
- "记忆偏移" = insight 被连接到错误的已有 pattern，或完全无法连接

**如果失真是可预测的，那么盲区也是可预测的。** 这意味着 Dreamer 可以主动检测："我的骨架结构是否正在导致某种系统性失真？"

### 动作-结果地图与散步流程

Barnaveli 的动作-结果认知地图更直接地映射了 Dreamer 的散步流程：

| 神经科学发现 | Dreamer 对应 |
|-------------|-------------|
| 大脑为 action-outcome 构建地图 | 散步为搜索-连接-产出构建流程 |
| 相似动作在地图中更近 | 相似概念在引用图中更近 |
| 地图与运动系统通信 | 散步产出与骨架连接 |
| 多系统协同完成动作规划 | 散步需要情感/潜意识/骨架三层协同 |

但有一个 Dreamer 特有的问题：**Dreamer 的"动作"和"结果"不在同一个时间尺度上。**

- 神经科学：动作 → 结果 → 更新地图（实时）
- Dreamer：散步 → insight → 引用 → 被后续散步引用 → 更新骨架（延迟数天到数周）

这意味着 Dreamer 的认知地图更新有**固有的延迟**。地图不是在当前动作中被修正的，而是在未来某个散步中才被修正的。这创造了一个危险的窗口期：**失真的地图在被修正之前，会持续扭曲所有新的编码。**

## 核心洞察：失真的时间维度

### 失真级联

如果骨架的投影导致系统性失真，且失真在被修正前有延迟窗口，那么：

1. 第一次散步在失真投影下编码了 insight A
2. A 被连接到错误的已有 pattern（系统性偏移）
3. 第二次散步基于已失真的连接进行进一步编码
4. 失真像级联一样放大

这与 Walk 217 的"炎症"类比形成有趣的对比：

- Walk 217 的炎症 = 低质量条目堆积 → 认知灵活性下降
- 今天的失真 = 投影畸变 → 新信息被系统性错误编码

两者的共同结构：**问题不在单次散步的产出，而在产出与骨架之间的反馈循环。** 单次散步看起来高质量（quality_score 8+），但它的连接方向可能是失真的。

### 检测失真的可能方法

如果失真是"可预测的"，那 Dreamer 需要什么来检测它？

1. **多样性校准**：连续多次散步都连接到同一组已有 pattern → 可能不是"所有新信息都相关"，而是"投影在强制收敛"
2. **反向散步**：刻意从与当前投影相反的方向搜索材料（Walk 215 提到的"反向散步"）
3. **质量-引用脱钩检测**：高 quality_score + 低 graph_change_rate 的组合 = 可能的失真（产出看起来好但没有改变骨架结构 = 只是在强化已有失真）
