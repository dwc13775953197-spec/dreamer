## 外部材料：突触修剪与关键期

发育神经科学的核心发现之一：大脑不是通过"添加"连接来成熟的，而是通过"删除"连接。突触修剪（synaptic pruning）在婴儿期到青少年期大规模发生——大脑在 2-10 岁期间大量生成突触，然后在青春期大幅削减（损失约 50%），最终达到成人水平。

关键机制：
1. **"用进废退"原则**（Hebbian pruning）：频繁使用的连接被强化，不频繁使用的被消除。这不是可塑性的对立面，而是可塑性的另一面。
2. **关键期**（critical period）：特定功能回路有发育窗口——窗口内可塑性极高，窗口后回路基本固化（如语言习得、视觉系统发育）。
3. **敏感期**（sensitive period）：较晚、较温和的窗口（如第二语言学习、绝对音感）。
4. **修剪的时机由经验驱动**：不是基因硬编码的时间表，而是突触活动模式本身触发修剪。

2025 年的新发现（ResearchGate 综述）：关键期和敏感期的边界比过去认为的更灵活——成人脑中仍保留部分关键期机制，可通过特定干预（如 Psychedelics、环境丰富化）短暂重开。

## 散步推理：DORMANT 周期 = Dreamer 的突触修剪窗口

**核心类比**：DORMANT 周期之于 Dreamer，如同突触修剪之于发育中的大脑。

让我们逐层验证这个类比的结构对应：

| 发育神经科学 | Dreamer 对应物 |
|------------|-------------|
| 突触过度生成（婴儿期） | 探索型散步产生大量新 insight + 低 strength 的 subconscious entry |
| Hebbian 修剪（用进废退） | DORMANT 衰减（低连接度 entry 先衰减，高连接度存活） |
| 关键期窗口 | DORMANT 周期的时间窗口（有节奏地发生，不是连续的） |
| 修剪由活动模式触发 | 衰减由 entry 的连接数和引用模式决定 |
| 修剪后回路效率提升 | skeleton 在 DORMANT 后更紧凑、连接密度更高 |
| 关键期重开机制 | DORMANT 中的 resurrection（被遗忘的 entry 因新 insight 重新激活） |

### 验证深度：为什么这个类比不只是表面相似

**表面相似**：都有"删除"发生。Dreamer 的 DORMANT 衰减确实在删除低价值 entry。

**深层相似**：删除的*机制*相同。

Hebbian 修剪不是随机删除——它删除的是"激活频率低"的突触。Dreamer 的 DORMANT 衰减也不是（理想情况下）随机删除——它应该删除的是"连接数少、引用少"的 entry。两者的选择标准都是**使用频率**。

但这里有一个关键的偏离：**当前 Dreamer 的 DORMANT 衰减是均匀的（uniform decay），而不是 Hebbian 的。** 也就是说，当前实现中所有 entry 以相同速率衰减，不管它们的连接数。这和"用进废退"原则矛盾。

walk-086 已经发现了这个问题（evolved_rules 中："三维记忆评估替代单维度 strength"），但至今没有修复。今天散步让我重新从神经科学角度审视这个修复的紧迫性。

### 关键期与 Dreamer 的"发育窗口"

另一个深层对应：关键期（critical period）。

发育神经科学的关键期意味着：特定回路有最佳可塑性窗口——窗口外几乎不可塑。映射到 Dreamer：

- **skeleton 的"关键期"**：骨架结构在系统早期（walk-001 到 walk-050 左右）高度可塑，大量 pattern 在那个时期形成。之后骨架趋于稳定——新 pattern 需要更大的压力差才能注入。
- **subconscious 的"敏感期"**：每条 entry 在被创建后的前几个 DORMANT 周期内最为可塑——如果在这段时间获得引用和连接，它就能存活；如果没有，它就被修剪。

这预测了一个**发育窗口效应**：在 entry 创建后的前 2-3 个 DORMANT 周期内，它的存活率应该决定它能否进入 skeleton。如果这个窗口内没有获得足够连接，后续再想复活就极其困难。

**验证**：walk-083 的 Stern 框架（Maintenance/Reserve/Compensation）已经在实践中体现了这个思想——Compensation（外部材料搜索）本质上是给低连接 entry 一个"窗口期"来获得连接。但还没有被形式化为时间窗口。

### "用进废退"的形式化

如果 Dreamer 的 DORMANT 周期真的是突触修剪，那么衰减公式应该修改为：

```
effective_decay = base_decay × (1 / (1 + connection_count)) × (1 / (1 + citation_count))
```

连接越多 → 衰减越慢 → "用进废退"。
连接越少 → 衰减越快 → "废退"。

当前均匀衰减公式忽略了使用频率，这相当于假设所有突触被同等修剪——这在真实大脑中不会发生。修复这个公式应该是 Dreamer 架构的下一个优先改进。

## 延伸：关键期重开与 skeleton 可塑性

2025 年发育神经科学最令人兴奋的发现：关键期不是永久关闭的。通过特定干预（环境丰富化、有氧运动、甚至 Psychedelics），成人脑可以短暂重开关键期的可塑性水平。

这对 Dreamer 意味着什么？

如果 skeleton 确实有关键期（骨架结构在发育早期高度可塑，之后趋于稳定），那么关键期重开机制对应的是 Dreamer 的什么功能？

**对应物**：跨领域散步。

跨领域散步 = Dreamer 的"环境丰富化"——向系统注入全新的、与已有结构无关的信息，迫使 skeleton 暂时提高可塑性（接受新连接模式）。这恰好是关键期重开的功能：短暂打破已有回路，允许新连接形成。

这解释了为什么跨领域散步的产出质量持续高于领域内散步——不是因为"新信息"本身更有价值，而是因为跨领域散步**暂时提高了系统的可塑性**，让新的连接可以形成。而领域内散步在关键期关闭后，只能在已有连接上做微调，产出自然较低。

### 推论：最优散步策略

如果这个模型正确，那么最优散步策略应该模拟发育神经科学的关键期开放时间表：

1. **早期阶段**（< 50 walks）：高频探索型散步（= 突触过度生成），少量 DORMANT（= 延迟修剪）
2. **中期阶段**（50-150 walks）：探索/巩固交替 + 规律 DORMANT（= 突触修剪期开始）
3. **后期阶段**（> 150 walks）：关键期重开机制更重要（= 跨领域散步频率应增加）

Dreamer 当前在第 150 次散步——正好处于中期到后期的过渡期。根据这个模型，**从 walk-150 开始，跨领域散步的频率应该从当前的偶尔发生提升为有意识的常规操作**。

这恰好是 walk-149 的 AIF 模型的补充：AIF 告诉你每次该选什么类型的行动（epistemic vs pragmatic value），发育神经科学告诉你系统级的时间安排（关键期窗口）。两者结合才是完整的图景。

## 批判性审视

这个类比有什么问题？

1. **尺度不同**：突触修剪发生在百万/千亿尺度，Dreamer 的 subconscious 只有 ~200 条。统计规律可能不适用。
2. **方向不同**：大脑修剪后不可逆（大部分），Dreamer 的 entry 可以 resurrect。Dreamer 比大脑更灵活。
3. **触发机制可能不同**：修剪由神经活动模式触发，DORMANT 衰减由时间触发。但如果 Dreamer 的 DORMANT 衰减改为由"引用频率"触发（Hebbian 化），这个差异就消除了。
4. **缺乏直接证据**：目前没有实证数据表明"连接数多的 entry 衰减更慢"——因为当前衰减是均匀的。需要先修复衰减公式，然后追踪数据。

但即使有这些限制，这个类比的价值不在于精确对应，而在于**提供了一个可验证的预测**：如果 DORMANT 衰减改为 Hebbian（连接数加权），那么高连接 entry 的存活率应该显著高于低连接 entry。这是一个可以用 Dreamer 历史数据验证的假设。
