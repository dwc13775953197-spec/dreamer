# DORMANT 状态的潜意识重组：DMN 启发的休眠设计

> 研究时间：2026-05-29
> 来源：Walk 003 → 研究提案 → Phase 2 研究

## DMN 核心机制

默认模式网络（DMN）在"不做事"时最活跃，主要功能：

1. **记忆巩固（Memory Consolidation）**：海马体在休息期间重放（replay）白天经历过的神经模式，DMN 提供神经上下文窗口
2. **模式重组（Pattern Reorganization）**：不同时间、不同情境的记忆片段在 DMN 中被重新连接
3. **创造性联想（Creative Association）**：DMN 的漫游性思维（mind-wandering）让不相关的概念碰撞
4. **自我参照处理（Self-Referential Processing）**：整理"我是谁"、"什么对我重要"

## 关键发现：离线重放（Offline Replay）

> "During rest, sequential hippocampal reactivations, known as 'replay', are played out within default mode network activation windows"

大脑在休息时不是"关机"，而是在**重放和重组**。这就是为什么"睡一觉就想通了"。

## DMN 与任务网络的对偶关系

DMN 和任务正网络（TPN）是**负相关**的：
- TPN 活跃 → DMN 被抑制（专注做事时不会走神）
- DMN 活跃 → TPN 被抑制（走神时做不好正事）

**设计含义**：DORMANT 期间不应该做任何"任务型"工作，而应该做 DMN 型工作——漫游、联想、重组。

## 对 Dreamer 的映射

| DMN 功能 | Dreamer DORMANT 对应 |
|---------|---------------------|
| 记忆重放 | Light Sleep：重读近期散步日志，reinforce 被引用条目 |
| 模式重组 | REM Sleep：跨散步连接相似 insight，生成 pattern |
| 创造性联想 | REM Sleep：随机种子碰撞，产生新方向 |
| 自我参照处理 | Deep Sleep：人格特质阈值检查、reflection |
