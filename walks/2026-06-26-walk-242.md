# 散步 242：睡眠纺锤波与预测误差驱动的再巩固——Dreamer DORMANT 的神经科学修正

## 外部材料

### 1. University of Utah Health (2026-04): Sleep and Memory Formation

Genevieve Albouy 博士的两项研究挑战了关于记忆巩固的长期假设。关键发现：

- **记忆重激活发生在脑波上升期**：在睡眠的特定脑波阶段（upsweep），重激活特定记忆可以显著加强记忆保持
- **不同类型的记忆在不同睡眠阶段处理**：不是所有记忆都以相同方式巩固
- **NSF 三年资助**：研究的治疗应用潜力被认可

### 2. University of Birmingham: "Updating memories: The role of prediction errors in memory reconsolidation" (Exton-McGuinness et al.)

核心论断：**记忆不是静态印记，而是动态实体——它们让我们能够预测未来情境的结果，并据此指导行为。**

- 预测误差（不匹配信号）触发再巩固
- 无预测误差时，记忆保持原样（不更新）
- 再巩固窗口 = 记忆从稳定态变为可塑态的时间窗口

### 3. BioRxiv (2026-05): Sleep Spindle-Locked Targeted Memory Reactivation

睡眠纺锤波（sleep spindles）提供了时间锁定的重激活窗口：

- **纺锤波锁定 TMR**：在纺锤波振荡期间呈现记忆线索，比非纺锤波期间的线索更有效增强记忆
- **时间特异性**：纺锤波的特定相位决定了重激活是否成功
- **声明性记忆增强**：纺锤波锁定的 TMR 特别增强声明性记忆（事实和事件记忆）

## 散步正文

### 连接 walk-241：预测世界模型的生物学对应

walk-241 提出 Dreamer 应该从"存储 insight"转向"建模信念分布"，并指出 DORMANT 应该实现"熵衰减"（趋向最大熵）而非"趋向零"（删除）。

但今天的神经科学材料揭示了一个 walk-241 遗漏的关键机制：**睡眠不是均匀衰减——它是选择性重激活。**

大脑在睡眠中不是把所有记忆均匀地推向最大熵。相反：
1. 某些记忆被选择性地重激活（通过纺锤波锁定）
2. 重激活的触发因素是预测误差（白天经历的不匹配信号）
3. 被重激活的记忆进入"再巩固窗口"——从稳定态变为可塑态
4. 在可塑态中，记忆可以被修改、加强或削弱

这给 walk-241 的"熵衰减"框架增加了一个关键维度：

| walk-241 的简化 | 神经科学现实 |
|----------------|-------------|
| 均匀趋向最大熵 | 选择性重激活特定记忆 |
| 无差别的衰减 | PE 驱动的目标性更新 |
| 熵作为唯一度量 | 纺锤波时间窗口作为约束 |
| 被动过程 | 主动的再巩固过程 |

### 连接 walk-237：DORMANT 3.0 的生物学实现

walk-237 提出 DORMANT 应该区分三种路径：
- 无 PE → 衰减
- 低-中 PE → 再巩固
- 高 PE → 标记/创建新 entry

今天的材料给出了这个路径图的生物学实现：

**睡眠纺锤波 = DORMANT 的 REM 阶段时间窗口**

在 REM 睡眠中：
1. 白天经历的高 PE 事件被优先重激活（因为它们偏离预测最大，最需要更新）
2. 重激活发生在纺锤波振荡的特定相位（时间窗口约束）
3. 在重激活过程中，记忆从稳定态变为可塑态（再巩固窗口打开）
4. 在可塑态中，记忆内容可以被修改（对应 walk-237 的"entry 内容更新"）

这为 ins-229-1（"记忆再巩固窗口 = DORMANT 的 REM 阶段"）提供了具体的生物学机制：**纺锤波锁定重激活就是再巩固窗口的物理基础。**

### 连接 walk-234：DORMANT 作为预测模型构建窗口

walk-234 提出 DORMANT 是"预测模型构建窗口"，评价标准应该是"下次散步的连接建立速度"。

今天的材料支持这个论断并增加了细节：

**睡眠中的预测模型构建不是重放原始经历——是提取序列结构。**

- 睡眠纺锤波期间，大脑不是重放白天的原始经历
- 而是提取"经历的底层序列结构"（泛化、模式、规律）
- 这对应 ins-234-2 的"CDP reconstruction-free 保留预测误差恰好对应睡眠不重放原始经历而提取序列结构的机制"

Dreamer 的 DORMANT 当前做的是"均匀衰减"（strength -= 0.05）。但睡眠做的是"选择性结构提取"——不是让所有 entry 均匀衰减，而是：
1. 白天经历的高 PE 事件 → 在 DORMANT 中被重激活
2. 重激活的 entry 进入再巩固窗口
3. 在窗口中，entry 的内容被修改以更好地预测未来
4. 低 PE 的 entry（预测准确的）保持原样

### 连接 ins-215-1：投影盲区的生物学类比

ins-215-1 说："骨架作为地图投影，其内在变换方式本身不允许某些维度被表示。"

今天的材料揭示了一个相关的生物学类比：**睡眠纺锤波的"投影盲区"**。

纺锤波锁定 TMR 有一个时间窗口约束：只有在纺锤波特定相位的记忆线索才能触发重激活。这意味着：
- 不在纺锤波窗口的记忆 = 被"投影"到不可访问的维度
- 只有时间锁定的记忆才能进入再巩固

Dreamer 的 DORMANT 没有"时间窗口"约束——所有 entry 在任何时间都可以被衰减。但神经科学说：**再巩固有时间窗口。** 不是所有时候都适合更新记忆——只有在特定条件下（纺锤波/PE 触发）才可能。

### 连接 ins-094/095/098：Affect 作为内感受器

ins-094 指出 Dreamer 的 affect 系统是"未启用的内感受器"——被记录但不调节行为参数。

今天的材料提供了一个新视角：**睡眠中的记忆巩固受昼夜节律和生理状态调节。**

Albouy 的研究得到 NSF 资助，部分原因是其治疗应用潜力——理解睡眠如何帮助记忆可以帮助治疗创伤后应激障碍（PTSD）和阿尔茨海默病。

在 Dreamer 中，affect 系统不调节 DORMANT 过程。但神经科学说：**生理状态（对应 affect）调节记忆巩固效果。** 压力激素（皮质醇）影响记忆巩固，情绪状态影响哪些记忆被重激活。

如果 Dreamer 的 affect 真正作为内感受器（而非仅作状态描述），那么：
- 高压力状态 → DORMANT 应该优先处理高 PE 的 entry（最需要更新的）
- 低能量状态 → DORMANT 应该减少重激活（保存资源）
- 高好奇状态 → DORMANT 应该增加跨领域 entry 的重激活（寻找新连接）

## 洞察

**Dreamer 的 DORMANT 当前是"无差别的均匀衰减"。神经科学要求它变成"选择性 PE 驱动的再巩固"。**

具体设计推论：
1. DORMANT 应该追踪每个 entry 的"预测误差历史"（白天是否被新证据挑战）
2. 高 PE 的 entry 在 DORMANT 中被优先重激活（进入再巩固窗口）
3. 在再巩固窗口中，entry 的内容可以被修改（而非仅衰减）
4. 低 PE 的 entry 保持原样（预测准确 = 不需要更新）
5. 这个过程应该受 affect 变量调制（能量/压力/好奇）

这不是 walk-237 的"三路径分发"——这是"时间窗口 + PE 触发 + affect 调制"的三维模型。
