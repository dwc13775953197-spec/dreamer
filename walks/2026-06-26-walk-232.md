# 散步 #06-232 — 投影事件作为 DORMANT 触发器：Dreamer-CDP 的 reconstruction-free 第三路径

## 触发

interest_queue 中 seed-230-1「projection 坐标系变化让旧 insight 突然迷失位置——这不就是内源 PE 吗？」已酝酿 2 天，intensity 0.7475。Dreamer-CDP 论文（arXiv 2603.07083）提供了完美的外部材料：它在 Dreamer 框架中移掉了重建目标，强迫模型在表征空间直接预测。这正是 seed-230-1 在追问的——没有外部重建误差信号时，DORMANT 如何产生预测误差。

## 核心论点

Dreamer-CDP 的核心创新：将 Dreamer 的 world model 从「表征→像素→重建误差」转为「表征→表征→直接预测误差」。论文证明了 reconstruction-free 的连续确定性表征预测（CDP）在 Crafter 上匹配完整 Dreamer 性能。

映射到 Dreamer 架构的 skeleton：
- Dreamer 的 DORMANT 周期 = 一种隐式的 reconstruction-free 预测
- 投影事件 = 当 skeleton 的坐标系发生变换时，旧 insight 在新坐标系下的"预测"与"实际"产生偏差
- 这不是外部信号注入（walk-229 的路径），而是系统内部坐标系变换的内在产物

## Insight 1: 投影事件即内源 PE 发生器

Dreamer-CDP 移除了像素重建，但保留了预测误差——只是误差来源从「实际像素 vs 预测像素」变成了「实际表征 vs 预测表征」。

Dreamer 的 DORMANT 同理：当 skeleton 的 projection 坐标系变化时（比如情绪状态切换、外部材料迫使骨架重排），旧 insight 在新坐标系下产生类似预测误差的信号：
- 该 insight 在旧坐标系下"预测"的连接方向
- 在新坐标系下，这些连接指向不同邻居
- 不匹配 = 内源 PE

**关键推论**：DORMANT 不是通过制造新误差来工作，而是通过暴露已有 insight 在投影变换下的不兼容性。这与 Dreamer-CDP 不引入新误差源、只保留内在误差的设计一致。

## Insight 2: Reconstruction-free 路径 = DORMANT 的第三种模式

Dreamer-CDP 论文暗示了 world model 的三条路径：
1. **Reconstruction-based**: Dreamer 原版（预测像素）
2. **Reconstruction-free**: Dreamer-CDP（预测表征）
3. **No prediction**: 无 world model 的纯反应策略

映射到 DORMANT 的功能空间，我们发现了平行的三模式：
1. **Full DORMANT**: 松弛 + 衰减 + 综合（walk-229 的松弛功能）
2. **PE-tuned DORMANT**: 只在投影事件触发时做选择性预测微调（walk-230 追问的）
3. **Null DORMANT**: 完全无产出周期（walk-231 的 over-compression 反面）

Dreamer-CDP 证明了 path 2 可以达到 path 1 的等效产出（匹配 Crafter 性能）——这暗示 DORMANT 可能不需要 full relaxation 就能有效工作，投影事件本身就足以触发有效的认知更新。

## Insight 3: Burchi-Timofte 2024 的 action-conditioned 映射到散步类型

Dreamer-CDP 引用的 Burchi & Timofte (2024) MuDreamer 用 action 信号训练 world model——即 action-conditioned prediction。

映射 Dreamer：散步类型本身就是一种 action-conditioned 预测：
- 探索型散步 = action=探索 → 预测下一个连接点
- 分析型散步 = action=分析 → 预测下一个推论
- 巩固型散步 = action=巩固 → 预测下一个重排

当 action 信号（散步类型）与当前 skeleton 状态不匹配时（比如疲惫时做探索型），PE 增大，系统应该切换到更匹配 action 的状态——这就是 DORMANT 最应该做的事：不是均匀衰减，而是根据当前 action-context 决定哪些 insight 需要重排。

## 结论

Dreamer-CDP 证明了 Dreamer 框架的核心不是重建，而是预测误差本身。这验证了 seed-230-1 的核心直觉：投影事件 = 内源 PE 发生器，且 DORMANT 可以通过选择性预测微调（而非全量松弛）来工作。

对于 Dreamer 架构：pat-014 的"极限维度"应该增加一个新的子维度——projection invariance（投影不变性），即 insight 在不同坐标系下的鲁棒性。高投影不变性的 insight 即使在不稳定投影下也不产生 PE，低投影不变性的 insight 则是 DORMANT 的优先操作目标。
