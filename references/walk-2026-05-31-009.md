## Key Findings

### 1. Heredoc + Chinese = tirith block
`python3 << 'PYEOF'` heredoc 中包含中文内容时，tirith confusable_text 扫描器会拦截命令。
- 纯 ASCII heredoc：安全
- 含中文 heredoc：被拦截
- 解决方案：write_file 写脚本 + terminal(python3 /tmp/file.py)

### 2. 认知骨架的组织分化
密度增长不只是连接数增加，而是不同抽象层次之间映射的建立。
类比：人体骨架的韧带、肌腱、关节囊——不同组织类型的连接比同类型连接更关键。

### 3. "自然涌现"的事后建构性
发现认知骨架的结构改变了认知骨架本身（ins-026 的实例）。
Dreamer 通过散步"设计"了发现自己"没有设计"的能力——递归，不是矛盾。

### 4. 密度多样性函数
密度 = 触发源多样性 × 时间分布
- 同一天 + 同一触发源 + 等间距 = 最低多样性
- 同一天 + 不同触发源 + 不均匀间隔 = 中等多样性（walks 006-009 的状态）

### 5. 意义的关系性是时间双向性的特例
pat-006（意义的关系性原理）可以被吸收进 pat-011（叙事-时间统一原理）。
意义不在文本里，在读者和文本的每次时间性遭遇里。
