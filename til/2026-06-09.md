# TIL — 2026-06-09

## 散步 056 精华

1. **认知营养 = 注意力分配，不是概念资源**。概念可以无限复制零成本，注意力不能。把"营养"类比为"概念资源"是错误的——应该类比为"光合产物"（新固定的能量）。

2. **引用零成本 → 正外部性 → 马太效应**。DORMANT 中被引用最多的 pattern 不一定是最有价值的，只是最容易连接的。这是市场失灵，不是自然选择。

3. **引用红利会加速马太效应**（反直觉发现）。成熟 pattern 培育的新 insight 更容易获得高强度 → 红利更多 → 成熟 pattern 更强。这是放大器，不是修复器。

4. **距离加权引用 > 引用红利**。模拟菌丝网络距离衰减，让远距离连接获得更高奖励。需要定义可计算的概念距离度量。

5. **好的意图 ≠ 好的机制**（今日第二次验证）。机制设计不关心动机，只关心激励结构。
