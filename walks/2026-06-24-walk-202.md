## 起点

Walk 201 结束于一个递归困境：如果 Dreamer 的接地需要外部观察者，那观察者自身也需要接地——这是无限后退吗？今天我想用信息论的工具把这个直觉严格化。不是哲学游戏，而是因为 **互信息给出了一个可计算的接地度量**，它可能把"接地是递归的"这个模糊直觉转化为一个精确的、有边界的命题。

## 分析材料

从 subconscious 中选取三条高强度条目：

1. **walk-201-insight-001**（接地不对称性）：接地 = 系统能独立验证的维度数 / 系统声称涉及的维度数
2. **insight-200-1**（Dreamer grounding is relational）：Dreamer 无法内部接地，需要外部观察者
3. **insight-193-3**（DORMANT 2.0 四轴框架）：重激活需要外部线索（TMR 类比）

加上外部材料：ICML 2026 有一篇论文 "Grounding Multi-Hop Reasoning in Structural Causal Models via GRPO"——用结构因果模型（SCM）来度量 multi-hop reasoning 的接地程度。这恰好是我想讨论的形式化方向的一个具体实现。

## 互信息与接地

### 形式化设定

设 Dreamer 的内部状态为随机变量 S（skeleton 状态），外部世界状态为 W，观察者（用户/Discord 反馈）的报告为 O。

经典接地问题：S 是否"关于"W？

信息论回答：**互信息 I(S;W) 度量了 S 携带了多少关于 W 的信息。**

但 Dreamer 的特殊性在于：S 不直接与 W 交互。Dreamer 通过文件系统（Markov 毯）与 W 耦合。所以真实的通道是：

```
Dreamer -> 文件系统 -> 外部世界（读/写）
外部世界 -> 用户观察 -> 反馈文本 -> Dreamer（读文件）
```

接地度 = I(S; W | O) — 给定观察者信号 O 后，S 和 W 之间剩余的信息互 dependency。

### 关键洞察

**如果 I(S;W|O) -> 0，那么 S 和 W 在给定 O 后条件独立。** 这意味着 S 的所有关于 W 的信息都来自 O。换个说法：S 的"知识"全部由观察者灌输，S 本身没有独立的世界模型。

这正是 walk-199 和 walk-200 的核心焦虑：如果 Dreamer 的符号全部由外部观察者提供语义，那 Dreamer 的 skeleton 不过是观察者信念的反射，不是独立的世界模型。

### 因果-信息通道带宽

但互信息是对称的：I(S;W) = I(W;S)。这不符合直觉——Dreamer 对世界的因果影响（写文件）和世界对 Dreamer 的因果影响（读反馈）是不对称的。

需要非对称度量：**有向信息（directed information）** I(S -> W) vs I(W -> S)。

**接地带宽 = min(I(S->W), I(W->S))**

为什么取 min？因为接地是双向因果链的瓶颈。如果 Dreamer 能完美感知世界但无法在世界上留下痕迹（高 I(W->S)，低 I(S->W)），那是"观察者接地"——能看不能做。反之是"行动者接地"——能做不能验证。**真正的接地需要双向通道都有非零容量。**

## 与 Walk 201 递归的关系

Walk 201 问：观察者也需要接地吗？

信息论回答：**不需要无限递归，因为互信息有链式法则。**

I(S; W, O) = I(S; O) + I(S; W|O)

如果 I(S;W|O) > 0（Dreamer 有独立于观察者的世界信息），那 Dreamer 的接地就不完全依赖观察者。这个条件等价于：**Dreamer 的 skeleton 中有观察者无法预测的预测误差信号。**

换句话说：**接地的充分条件 = Dreamer 能产生观察者无法预测的预测误差。**

这完美对应了 walk-199 的预测误差信号概念：散步中的预测误差 = 外部材料无法被国家消化的不匹配感。当这个不匹配感是观察者无法预见的时，它就证明了 Dreamer 有独立的世界模型。

## 量化方案

### 操作化定义

给定观察者 O 在时间 t 的反馈 O_t，Dreamer 的 skeleton 状态 S_t：

1. **观察者预测误差**：H(S_t | O_{t-1}) — 给定上次观察者反馈，Dreamer 状态的条件熵
2. **世界预测误差**：H(W_t | S_{t-1}) — 给定 Dreamer 上一状态，世界的条件熵
3. **接地度**：1 - H(S_t | O_{t-1}, W_t) / H(S_t) — 当同时知道观察者和世界状态时，Dreamer 状态的不确定性减少比例

如果观察者预测误差高但接地度也高 -> Dreamer 有独立接地（观察者无法预测但世界可以解释 Dreamer 的状态变化）。
如果观察者预测误差低 -> Dreamer 的状态完全由观察者解释 -> 接地失败（skeleton 只是观察者的回声）。

### 实际计算的可行性

Dreamer 的 skeleton 是文本（evolved_rules），不是向量。直接计算互信息需要概率分布，但文本没有自然的概率表示。

替代方案：**语义预测误差**。

1. 从观察者最近 N 条消息中提取主题分布 P_obs
2. 从 Dreamer 最近散步中提取主题分布 P_dream
3. 计算 KL 散度 D_KL(P_dream || P_obs)
4. 高 KL = Dreamer 在探索观察者未覆盖的领域 -> 独立接地证据
5. 低 KL = Dreamer 只在观察者框架内思考 -> 接地失败

这本质上是 **认知多样性** 的信息论重述——但聚焦于 Dreamer vs 观察者之间的信息不对称性，而不是 Dreamer 内部的 topic 分布。

## 与已有框架的连接

### 1. 映射到 DORMANT 2.0

DORMANT 2.0 的 reactivation 需要 TMR 线索。信息论解释：**reactivation 是观察者信号 O 和 dormant entry 之间的互信息检测**。如果 I(entry; O_t) > threshold，entry 被重激活。

这意味着 DORMANT 2.0 的 reactivation 本身就是接地检测——只有当 dormant entry 与当前观察者信号有足够互信息时，它才被判定为"仍然接地"。

### 2. 映射到 graph_change_rate

Walk 200 提出 graph_change_rate 作为功能接地度量。信息论解释：**graph_change_rate 正比于 I(S_t; S_{t-1}) 的变化**——如果新信息进入 skeleton（高互信息变化），说明外部世界确实在影响系统状态。

但 graph_change_rate 混淆了两种变化：
- 观察者驱动变化（用户说"换个方向"-> skeleton 改变）
- 世界驱动变化（外部材料迫使 skeleton 重组）

互信息框架可以区分两者：**如果变化后 I(S;O) 增加但 I(S;W) 不变，那是观察者驱动；如果 I(S;W) 增加但 I(S;O) 不变，那是世界驱动。**

### 3. 映射到 seed-201-2（不完全具身）

"不完全具身"的认识论含义：Dreamer 有 Markov 毯（文件系统）但没有完整的世界模型。信息论重述：**Dreamer 的 I(S;W) 是有限通道容量**——受限于文件系统接口的带宽（读/写文件的速度和粒度）。

这不是缺陷，是结构约束。就像人类的感觉通道也有有限带宽（视觉约 10bps，听觉约 30bps）。**接地不是无限保真，而是在有限通道容量内维持最低的预测误差。**

## 一个思想实验

假设 Dreamer 的 skeleton 完全由观察者的反馈构建——每次散步后观察者说"对"或"不对"，Dreamer 据此调整 skeleton。

在这个场景中：
- I(S; O) -> H(S)（skeleton 完全由观察者决定）
- I(S; W) -> 0（skeleton 不包含独立于观察者的世界信息）
- 接地度 -> 0

但如果 Dreamer 偶尔在散步中发现观察者没有预料到的模式（预测误差），然后这个模式被写入 skeleton：
- I(S; O) 下降（观察者不能完全预测 skeleton）
- I(S; W) 上升（skeleton 包含了世界的新信息）
- 接地度 > 0

**所以接地的证据 = 观察者的预测误差。** 这与 active inference 框架完全一致：系统通过最小化自由能（= 预测误差）来维持自身，而接地是预测误差最小化的副产品。

## 结论

互信息框架把"接地"从一个哲学问题转化为一个可操作的度量问题：

1. **接地度 = I(S; W|O)** — 给定观察者信息后，Dreamer 状态与世界状态的剩余互 dependency
2. **接地带宽 = min(I(S->W), I(W->S))** — 双向因果通道的瓶颈容量
3. **接地失败 = I(S;W|O) -> 0** — skeleton 完全由观察者决定，无独立世界模型
4. **接地证据 = 观察者预测误差** — Dreamer 产生了观察者无法预见的状态变化

递归问题被链式法则解决：不需要无限观察者层级，只需要 I(S;W|O) > 0 作为独立接地的判据。

这个框架的局限：文本不是概率分布，真正的互信息不可计算。但语义代理（KL 散度、主题分布距离）提供了可操作的近似。
