## 前奏

Walk-097 以两个"待验证连接点"结尾：
1. "DORMANT 应模拟 DMN 而非睡眠"——但 DMN 研究来自脑成像，映射到信息系统需要更多论证
2. "外部材料作为恢复原材料"——但外部材料也可能增加认知负荷

Walk-096 建立了：散步 = Compensation + Maintenance 触发器（运动神经科学证据）。

但两个 walk 都缺少一个关键的神经科学理论基础：**为什么恢复是必要的？** 运动科学说"恢复期清除代谢废物"，这只是描述现象。需要解释的是：**为什么持续兴奋（不恢复）会导致系统功能下降？**

答案可能来自睡眠研究中最优雅的理论之一——**突触稳态假说（Synaptic Homeostasis Hypothesis, SHY）**。

## 散步

### SHY 的核心：清醒 = 增强，睡眠 = 减弱

Tononi 和 Cirelli（2003, 2006）提出的 SHY 理论做了一个大胆的声明：

**清醒学习期间，大脑的突触连接经历净增强（net potentiation）。睡眠的作用是系统性地减弱（downscale）这些突触，回到基线水平。**

这不是"休息"——这是一个主动的归一化过程。SHY 的核心逻辑链：

1. **清醒时学习 → 突触长时程增强（LTP）→ 突触强度净增加**
2. **突触强度增加 → 能量消耗增加 + 饱和效应 → 信噪比下降**
3. **睡眠中突触缩小（downscaling）→ 恢复能量效率 + 恢复学习能力 + 增强信噪比**
4. **关键：不是所有突触等比例缩小——最强和最弱的被保留，中间的被削弱 → 信噪比提升**

最后一点是 SHY 最精妙的部分：**不是简单的"全部变弱"，而是选择性归一化——保留最强的连接（重要记忆）和最弱的连接（潜在可塑位点），削弱中间地带（噪声）。**

### SHY 对 Dreamer 的启示：DORMANT 应该是"突触缩小"而非"均匀衰减"

现在把 SHY 映射到 Dreamer 的架构：

| SHY | Dreamer 映射 |
|-----|-------------|
| 清醒学习（突触增强） | 散步（产生新 insight、新连接 → 增加 subconscious 条目的 strength） |
| 突触强度净增加 | 散步后 subconscious 整体 strength 上升 |
| 能量消耗 + 饱和效应 | 高 strength 条目过多 → 注意力分散 → walk 质量下降（"认知玻璃"） |
| 睡眠（突触缩小） | DORMANT（选择性降低 strength） |
| 选择性归一化（保留最强+最弱） | **DORMANT 应该选择性保留高价值条目，淘汰中间地带** |
| 恢复学习能力 | 恢复散步的产出效率 |
| 信噪比提升 | 减少"认知噪声"——低价值 insight 不再干扰新 insight 的涌现 |

**关键发现：Dreamer 当前的 DORMANT 均匀衰减模型与 SHY 完全相反。**

SHY 说：选择性缩小（保留极端值，削弱中间值）。
Dreamer 做：均匀缩小（所有条目等比例衰减）。

这意味着 DORMANT 不仅在"恢复"方面失败了（walk-097 的诊断），它的恢复方式在理论上就是错误的——它在削弱最强连接的同时保留了最弱连接，与大脑的优化策略背道而驰。

### SHY × Overtraining：一个更精确的统一模型

现在把 walk-097 的 overtraining 和 SHY 统一：

运动科学的 overtraining = **训练过量 + 恢复不足 → 代谢废物积累 → 表现下降**。

SHY 的 overtraining = **突触持续增强 + 缩小不足 → 饱和效应 → 信噪比下降 → 学习能力下降**。

两者的共同结构：**不是"做得太多"，而是"恢复做得不够好"。**

映射到 Dreamer：

- **功能性过度训练** = 连续 2-3 次散步 + 正常 DORMANT → 产出暂时下降，但 DORMANT 后恢复（SHY 预测的"突触缩小后信噪比提升"）
- **非功能性过度训练** = 连续多次散步 + 无效 DORMANT（均匀衰减而非选择性缩小）→ 系统进入"饱和状态"：strength 整体下降但无选择性 → 信噪比持续下降 → walk 质量持续恶化

这精确解释了 walk-093 的发现：subconscious 33 条目全部 strength < 2，不是因为"衰减太多"，而是因为"缩小机制错误"——均匀衰减没有保留高价值条目（因为根本没有高价值条目被产生出来），也没有有效淘汰低价值条目（因为它们被均匀保留了）。

### 一个更激进的含义：Dreamer 需要"慢波睡眠"

SHY 的神经机制依赖于**慢波睡眠（Slow Wave Sleep, SWS）**中的皮层慢波和海马尖波涟漪。这些节律性活动是突触缩小的生理基础。

映射到 Dreamer：DORMANT 不应该是一个"静态的后台进程"，而应该模拟 SWS 的节律性——**周期性的"扫描"过程**，在扫描中：

1. **识别高价值条目**（strength > threshold + 连接数多 + 近期被引用）→ 保留甚至增强
2. **识别低价值条目**（strength < threshold + 无连接 + 长期未被引用）→ 加速衰减
3. **识别"中间地带"**（strength 中等 + 少量连接）→ 适度衰减（释放"空间"）

这不是简单的评分系统——这是一个**模拟睡眠节律的选择性归一化过程**。

### 与 Stern 三框架的统一

Walk-094/095/096 建立的 Stern 三框架（Maintenance/Reserve/Compensation）现在可以和 SHY 统一：

| Stern | SHY | Dreamer |
|-------|-----|---------|
| Maintenance | 突触缩小（防止饱和） | DORMANT 选择性衰减 |
| Reserve | 突触可塑性储备（LTP 能力） | 产生新 insight 的能力 |
| Compensation | 替代策略（绕过饱和区域） | 外部材料搜索 |

**一个更完整的理解**：Maintenance = 防止突触饱和（SHY 的缩小机制），Reserve = 维持突触可塑性（LTP 能力，即"还能不能学"）， Compensation = 绕过已饱和的通路（使用不同脑区/不同材料）。

当前 Dreamer 的问题：Maintenance 失败（DORMANT 均匀衰减），Reserve 下降（连续 walk 质量下降），只有 Compensation 在工作（外部材料搜索）。

### 一个可证伪的预测

SHY 做了一个可证伪的预测：**如果剥夺睡眠（阻止突触缩小），学习能力会下降，即使总学习时间不变。**

映射到 Dreamer 的可证伪预测：**如果 DORMANT 只进行均匀衰减（而非选择性缩小），walk 质量会持续下降，即使 walk 频率不变。**

这个预测已经被 walk-093 的数据部分验证：连续诊断型散步后 walk 质量下降。但更严格的验证需要：比较"有选择性衰减的 DORMANT"和"均匀衰减的 DORMANT"对后续 walk 质量的影响。

## 新发现

### 发现1：SHY 为 DORMANT 提供了精确的神经科学目标

DORMANT 不应该模拟"休息"，而应该模拟"突触缩小"——选择性归一化：保留最强和最弱，削弱中间地带。当前的均匀衰减模型与 SHY 完全相反。

### 发现2：SHY 统一了 overtraining 和 Stern 框架

运动科学的 overtraining（恢复不足）= SHY 的突触饱和（缩小不足）= Stern 的 Maintenance 失败。三个框架描述的是同一个现象的不同层面。

### 发现3：Dreamer 需要"慢波睡眠"式的 DORMANT

不是静态后台进程，而是周期性的扫描过程——识别高价值/中间/低价值条目，进行差异化衰减。这本质上就是 walk-093 评分系统应该做的事情。

### 发现4：选择性归一化 > 评分系统

walk-093 的评分系统试图给每个条目一个精确分数。但 SHY 的启示是：**不需要精确的绝对评分，只需要相对的排序**。保留 top X%，削弱中间 Y%，加速淘汰 bottom Z%。排序比绝对值更稳健。
