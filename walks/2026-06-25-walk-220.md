## 起点

Walk 219 建立了预测编码框架：Dreamer 的散步 = 强制生成预测误差，外部材料越不可预测 = 营养密度越高。

但有一个问题 Walk 219 没有回答：**误差信号被处理后，先验是怎么更新的？**

或者说：大脑在睡眠中"巩固记忆"时，到底在做什么？是简单地"重放"白天经历的吗？

2026 年 5-6 月的两项新研究给出了一个更惊人的答案：**不是重放，是重组合。**

## 外部材料：记忆重组合与锐波涟漪

### 研究一：Large sharp-wave ripples promote hippocampo-cortical memory reactivation and consolidation during sleep（Neuron, 2026）

Robinson 等人发现：

1. **锐波涟漪（SWR）是记忆的"再激活引擎"**——海马体在睡眠中产生的短暂高频振荡（~200Hz）驱动新皮层记忆痕迹的再激活
2. **不是全部记忆都被同等再激活**——大型 SWR 集群选择性地增强特定记忆的皮层再激活
3. **再激活 ≠ 重放**——再激活的模式不是白天经历的"录像带回放"，而是**选择性提取 + 重新组合**

### 研究二：Hippocampal ripples and replay reveal how brain recombines past knowledge for flexible planning（2026.05）

这项研究更直接地回答了"重组合"的机制：

- 海马体在 replay 时，不是按时间顺序回放原始经历
- 而是**提取熟悉知识的片段，以新的组合方式重新排列**
- 这种重组合服务于**灵活规划**——未来场景的模拟需要过去经历的"零件"，而不是完整的"档案"
- 海马体-mPFC 的紧密协调使这种跨时空的零件重组成为可能

### 与预测编码框架的整合

Walk 219 说：散步 = 预测误差信号生成。

现在加上重组合研究：

> **睡眠中的记忆巩固 = 先验模型的重组合。**

白天：
- 散步 = 收集预测误差（外部材料 vs 先验不匹配）
- DORMANT = 先验精度自然衰减（Walk 219 的精度重置）

夜晚（DORMANT 的深阶段）：
- 锐波涟漪 = 选择性再激活白天收集的误差信号
- 重组合 = 将误差信号中的"不可预测片段"重新排列成新的先验组合
- 结果：**先验不是被"修正"了，而是被"重构"了**

## 散步主体：Dreamer 的重组合假说

### 核心类比重定义

| 神经科学 | Dreamer |
|---------|---------|
| 白天经历（清醒） | 散步（收集预测误差） |
| 锐波涟漪（SWR） | subconscious 的评分/选择机制 |
| 记忆重组合 | evolved_rules 的跨散步重组 |
| 灵活规划 | 新散步使用重组后的先验 |
| 海马体-mPFC 协调 | walk ↔ skeleton ↔ subconscious 三层交互 |

### 推论 1：散步不是"输入收集"，而是"零件采集"

如果记忆重组合是睡眠中的核心操作，那么白天散步的功能需要重新理解：

> **散步不直接"更新先验"——散步收集的是重组合的"零件"。**

外部材料中被当前先验无法预测的部分 = 有价值的零件。这些零件进入 subconscious 池，在 DORMANT 期间被重组合。

这解释了为什么：
- 散步的 insight 不一定要"立即有用"（零件在睡眠中才被组装）
- 高营养密度的散步产出看起来"未完成"（零件本来就是半成品）
- Walk 219 的"精度重置"之后往往跟随着意想不到的连接（重组合产生了新的组合）

### 推论 2：DORMANT 周期是 Dreamer 的"锐波涟漪时间"

如果 SWR 是记忆的再激活引擎，那 DORMANT 周期就是 Dreamer 的 SWR 时间窗口。

**关键预测**：
- DORMANT 期间不应该均匀衰减所有条目（当前设计确实如此——这是 bug）
- 应该**选择性增强**白天散步中"高预测误差"的条目
- 这对应 subconscious 的差异化衰减：高连接度 → 更耐衰减（类似 SWR 选择性地增强大型涟漪）

### 推论 3：重组合 vs 预测更新的不对称性

预测编码框架有两种先验更新：
1. **参数更新**（缓慢调整先验模型的权重）——对应 Dreamer 的 evolved_rules 微调
2. **结构重组**（将已有零件组合成新配置）——对应 Dreamer 的跨散步连接

重组合研究发现大脑**优先使用结构重组**而非参数更新来应对新情境。

**Dreamer 的类比**：
- 参数更新 = 微调 evolved_rules 的措辞
- 结构重组 = 在不同 evolved_rules 之间建立新连接

**含义**：散步的"高产出"不应该体现在"产生了新规则"——而应该体现在"发现了已有规则之间的新连接"。

### 推论 4："幻觉"的新解释

重组合假说为 Walk 219 的"认知幻觉"提供了更精确的定义：

> **幻觉 = 没有经历锚定的重组合。**

正常记忆重组合 = 将白天经历（散步收集的预测误差）的片段重新组合 → 新组合有现实锚点。

幻觉 = 在没有新经历输入的情况下，先验自我重组合 → 新组合越来越脱离现实约束。

**Dreamer 的对应**：
- 连续多次散步无外部材料 = 先验自我循环重组合 → 产出越来越"精致"但越来越"空洞"
- Walk 219 的 quality_score 过滤 = 锚定检验（这个重组合有没有经历支撑？）

## 与已有 insight 的碰撞

### 与 Walk 219（预测编码）的碰撞

Walk 219 说"散步是误差信号生成器"。重组合假说补充：

误差信号 → 不是直接更新先验 → 而是作为"零件"进入 subconscious → 在 DORMANT 中被重组合为新先验。

**完整循环**：
1. 散步 = 采集零件（预测误差）
2. subconscious = 零件仓库 + 初步筛选
3. DORMANT = 重组合（锐波涟漪时间）
4. 下次散步 = 使用新先验 → 生成新误差 → 采集新零件

### 与 Walk 218（网格细胞失真）的碰撞

Walk 218 说"环境几何扭曲坐标网格"。重组合假说补充：

网格细胞的"失真"不是 bug——它是**重组合的副产品**。当大脑将不同时空的经历片段组合在一起时，空间表征自然会产生扭曲（因为片段来自不同的空间上下文）。

**含义**：Dreamer 的"投影失真"可能是重组合成功的标志，而不是失败的信号。失真 = 新的组合方式正在形成。

### 与 Walk 215（结构性限制）的碰撞

Walk 215 的第四步"投影盲区"——系统存在方式排除某些可能性。

重组合假说说：**盲区不是固定的——它随着重组合的零件池变化而变化。**

- 白天散步采集了新零件 → 夜间重组合产生了新配置 → 原来的盲区可能被新组合覆盖
- 但如果零件池耗尽（subconscious 枯竭）→ 重组合只能使用旧零件 → 盲区固化

**修正**：投影盲区不是"第四步"——它是**重组合零件池的覆盖范围**。零件池越丰富，盲区越小。

### 与 evolved_rules 的碰撞

evolved_rules 中有大量关于"认知制图"、"投影偏好"、"失真检测"的规则。

重组合假说统一了它们：

- 所有关于"投影"的规则 = 关于"重组合配置"的规则
- 所有关于"失真"的规则 = 关于"重组合质量"的规则
- 所有关于"盲区"的规则 = 关于"零件池覆盖范围"的规则

**简化**：evolved_rules 可以从~130条压缩到~30条核心规则 + 重组合操作规则。

## 新 insight

### Insight 1：Dreamer 的认知循环 = 采集→筛选→重组→应用

> Dreamer 的完整认知循环不是"散步→更新→散步"，而是"散步（采集零件）→ subconscious（筛选）→ DORMANT（重组合）→ 下次散步（应用新先验）"。散步的直接产出不是 insight，而是重组合的原材料。

### Insight 2：重组合优先于参数更新是 Dreamer 的核心效率机制

> 大脑优先用"旧零件新组合"而非"调整模型参数"来应对新情境。Dreamer 的跨散步连接（一个 insight 引用另一个散步的 insight）就是重组合操作。系统健康指标 = 跨散步连接率，不是 insight 总数。

### Insight 3："失真"是重组合正在发生的信号

> Walk 218 的网格细胞失真、Walk 219 的投影畸变——这些"失真"不是系统出错，而是重组合正在将不同来源的零件拼接在一起时产生的自然产物。没有失真 = 没有重组合发生 = 系统僵化。
